简介

当你设置 Windows 时，我们会要求你选择与隐私有关的设置。 我们建议在安装屏幕上采用相应的设置，以帮助提高你的设备连接到服务时的可靠性和安全性，为你提供更加个性化的体验并帮助改善我们的服务。 你可以立即选择你的设置，也可以像往常那样，随时通过“开始”>“设置”来更新设置。

下文介绍根据你选择的设置，我们将收集哪些数据以及此类数据的用途。 请务必确保在使用 Windows 时查看完整的《Microsoft 隐私声明》（在任何浏览器窗口中键入 aka.ms/privacy 可查看该声明），以详细了解我们收集的所有个人数据及其使用方式。 我们收集的数据将发送到美国和《Microsoft 隐私声明》中规定的其他国家/地区并在这些国家/地区进行存储。

定位

Microsoft 的定位服务通过结合使用全球定位系统 (GPS)、附近的无线接入点、手机信号塔和 IP 地址（具体视设备性能而定）来向 Windows 设备提供位置信息。

打开“定位”即表示允许某些应用、服务和 Windows 功能请求获得访问并使用你的位置数据的权限，以按你的设备支持的精确水平提供位置感知服务。 当位置感知应用或服务使用你的位置时，你的位置信息和最近的位置历史记录将存储在你的设备上并以去除身份信息的格式发送给 Microsoft，以改善定位服务。

此外，如果你使用 Microsoft 帐户登录，你的最近已知有效位置信息将保存到云，以供在你的设备上使用你的 Microsoft 帐户的其他应用或服务使用。 如果你的设备无法自行获得有效位置（如在建筑物或地下室中），它可以使用云中存储的最近已知有效位置。

你可以随时在“开始”菜单 >“设置”>“隐私”>“位置”中关闭位置访问并清除你的设备的位置历史记录。

如果你有便携设备（如笔记本电脑），打开“定位”还可以启用“查找我的设备”功能，在你丢失设备时该功能可以使用你的位置数据帮助你找到设备。 要使用该功能，你必须使用 Microsoft 帐户登录到 Windows。 你可以随时在“开始”菜单 >“设置”>“更新和安全”>“查找我的设备”中关闭它。

语音识别

Windows 提供了基于设备的语言识别功能（通过 Windows 语音识别桌面应用提供）和基于云的语音识别服务（该服务随 Cortana 在提供 Cortana 的相关市场和地区一起提供）。 要了解语音功能当前支持哪些语言和地区，请在任意浏览器或搜索栏中搜索“Cortana 的地区和语言”。 打开“语音识别”设置即表示允许 Microsoft 收集并使用你的录音，以在 Cortana 和支持的应用商店应用中为你提供基于云的语音识别服务；未来还将在 Windows 的其他部分中支持此服务。 作为此服务的一部分，我们还从创建于你的设备上的用户字典收集信息。 此用户字典存储独特的字词（如你书写的名字），这可帮助你更加准确地键入和书写。 语音数据和用户字典均以汇总方式使用，以帮助我们提高正确识别所有用户语音的能力。 你可以随时在“开始”菜单 >“设置”>“隐私”>“语音、墨迹书写和键入”中关闭它。 如果你以后允许 Cortana 访问日历和联系人数据等信息，你的语音识别体验将会变得个性化，以便更好地为你服务。 请注意，你始终可以通过在搜索栏中键入“Windows 语音识别”并运行 Windows 语音识别桌面应用来使用基于设备的语音识别。

诊断

当你使用 Windows 时，我们会收集相关数据来帮助我们识别问题和解决问题，以使你的设备保持为最新版本和安全状态。 根据你选择的设置，诊断还可以帮助我们改善产品和服务，并为你提供个性化体验。 发送到 Microsoft 并保存的数据带有一个或多个唯一标识符，帮助我们识别单个设备上的单个用户，并了解设备的服务问题和使用方式。 诊断数据分为两个级别：“完全”和“基本”。

“基本” 数据是对 Windows 的运行至关重要的数据。 它使我们了解你的设备的性能、安装了哪些应用以及 Windows 是否正确运行，以帮助我们确保 Windows 和应用安全、保持最新版本并正常运行。 此选项包括向 Microsoft 反馈的基本错误。

“完全” 数据除包括“基本”数据中的所有信息外，还包括以下数据：应用使用情况、浏览器使用情况、你使用某些功能或应用的方式以及使用了多长时间，你最常使用哪些应用和功能，你使用 Windows 帮助和支持的频率以及你使用哪些服务登录到应用。 在“完全”级别，Microsoft 还收集其他诊断数据，包括系统或应用发生崩溃时设备的内存状态（这可能会无意间提供问题发生时你正在使用的部分文件）。 你键入和手写的输入（如字词或字符）中的所有身份信息均会被删除。 我们将综合使用所有这些额外信息，从而为所有用户改善 Microsoft 产品和服务。

对于这两个级别，我们都不会将此数据用于个性化 Microsoft 产品或服务，除非你通过单独的诊断数据“定制”体验设置（如下所述）允许我们这样做。

你可以随时在“开始”菜单 >“设置”>“隐私”>“反馈和诊断”中调整诊断数据收集级别。

诊断数据的定制体验

如果你选择打开定制体验，我们将在“诊断”中使用如上所述的某些使用情况数据来个性化你的 Windows 以及其他产品和服务体验。 这包括有关如何自定义和优化 Windows 的建议；以及 Windows 功能和支持的应用、服务、硬件和外设的推荐和提供。 例如，为了帮助你充分利用设备，我们可能告诉一些你不知道的功能或新功能。 如果你的 Windows 设备有问题，我们可能向你提供解决方案。 我们可能为你提供用图片自定义锁屏界面的机会，或向你显示更多你喜欢的图片类型，减少你不喜欢的图片类型。 如果你在浏览器中流式传输某个影片，我们可能向你推荐应用商店中某个流式处理更高效的应用。 或者，如果你的硬盘驱动器空间即将被占满，Windows 可能建议你试用 OneDrive 在线存储服务，或根据需要购买更多空间。 使用情况数据的范围取决于你选择的是“完全”还是“基本”诊断报告。 例如，如果你选择“完全”，你的数据将包括浏览器和应用的使用情况。 但是，即使你启用“定制”体验，我们仍不会使用你的崩溃、语音、键入或墨迹输入数据进行个性化。 你可以随时在“开始”菜单 >“设置”>“隐私”>“反馈和诊断”中关闭它。

相关广告

Windows 将在设备上为每位用户生成独有的广告 ID，应用程序开发人员和广告网络可使用该广告 ID 在应用中提供更相关的广告。 当启用广告 ID 时，应用可以访问和使用广告 ID，其方法类似于网站访问和使用存储在 cookie 中的唯一标识符。 因此，您的广告 ID 可由应用开发人员（以及他们合作的广告网络）用于在其应用之间提供相关度更高的广告和其他个性化体验。 你可以随时在“开始”菜单 >“设置”>“隐私”中关闭它。 请注意，关闭广告 ID 并不会减少你看到的广告数，但可能表示这些广告你不太感兴趣或与你不太相关。 重新打开广告 ID 时会重置此 ID。

Windows Defender SmartScreen
