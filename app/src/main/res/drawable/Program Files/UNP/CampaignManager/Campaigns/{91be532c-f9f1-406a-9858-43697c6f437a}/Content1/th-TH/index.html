เราได้ประสบปัญหา เราจะลองอีกครั้งในอีกสักครู่

รหัสข้อผิดพลาด:

มีใน Windows 10 เร็วๆ นี้

สิ่งหนึ่งที่ดีที่สุดเกี่ยวกับ Windows 10 ก็คือ มีประสิทธิภาพที่ดีขึ้นอย่างต่อเนื่อง คุณลักษณะใหม่ ความปลอดภัยที่ได้รับการอัปเดต ทันสมัยตลอดเวลาได้อย่างง่ายดาย ยอดเยี่ยมยิ่งขึ้น เมื่อเปิดใช้งานการอัปเดตอัตโนมัติ คุณจะได้รับคุณลักษณะใหม่ของ Windows 10 และการปรับปรุงความปลอดภัยทันทีที่พร้อมใช้งาน โดยไม่ต้องเสียค่าใช้จ่ายเพิ่มเติมตลอดระยะเวลาการใช้งานของอุปกรณ์*

นวัตกรรมใหม่ที่กำลังจะมาถึงก็คือ การอัปเดต Windows 10 สำหรับนักสร้างสรรค์ การอัปเดตนี้จะช่วยเพิ่มประสิทธิภาพด้านความปลอดภัยล่าสุด รวมถึงมอบคุณลักษณะใหม่ๆ ที่ยอดเยี่ยมมากมาย เช่น

  • 3 มิติใน Windows 10 มีแอปพลิเคชันระบายสี 3D สำหรับการสร้างสรรค์และการแชร์ในระดับใหม่
  • การเล่นเกมบนพีซี พร้อมเกม 4K โหมดเกม และการกระจายข้อมูลเกมที่ติดตั้งในตัว
  • จอแสดงผลแดชบอร์ดความปลอดภัย และตัวให้คำแนะนำสถานภาพของอุปกรณ์

ประสบการณ์การใช้งาน Windows 10 ของคุณจะดียิ่งขึ้น!

คุณลักษณะบางอย่างจะขึ้นอยู่กับฮาร์ดแวร์ *อาจมีค่าบริการเมื่อเชื่อมต่ออินเทอร์เน็ต

บทนำ

เมื่อคุณตั้งค่า Windows เราจะขอให้คุณเลือกการตั้งค่าที่สัมพันธ์กับความเป็นส่วนตัวของคุณ เราได้แนะนำการตั้งค่าที่พิจารณาแล้วว่าเหมาะสมไว้ที่หน้าจอการติดตั้งเพื่อช่วยให้คุณตั้งค่าอุปกรณ์ของคุณให้น่าเชื่อถือและปลอดภัยยิ่งขึ้นเมื่อเชื่อมต่อกับบริการต่างๆ เพื่อให้คุณได้รับประสบการณ์การใช้งานที่เป็นแบบของคุณมากขึ้น และช่วยปรับปรุงบริการต่างๆ ของเรา คุณสามารถเลือกการตั้งค่าได้เลยตอนนี้ หรือจะอัปเดตการตั้งค่าเหล่านี้ได้ทุกเมื่อตามปกติ โดยไปที่ เริ่มต้น > การตั้งค่า

ข้อมูลด้านล่างนี้จะอธิบายให้ทราบว่าเราจะเก็บรวบรวมข้อมูลอะไรบ้าง และจะนำข้อมูลที่เก็บรวบรวมได้ไปใช้อย่างไร โดยจะขึ้นอยู่กับการตั้งค่าที่คุณเลือก โปรดตรวจสอบให้แน่ใจว่าคุณได้ตรวจทานคำชี้แจงสิทธิ์ส่วนบุคคลฉบับเต็มของไมโครซอฟท์แล้ว เพื่อดูข้อมูลเพิ่มเติมเกี่ยวกับข้อมูลส่วนบุคคลทั้งหมดที่เราเก็บรวบรวม และจะนำข้อมูลดังกล่าวไปใช้อย่างไรเมื่อคุณใช้งาน Windows (พิมพ์ aka.ms/privacy ลงในหน้าต่างเบราว์เซอร์ใดก็ได้เพื่อดำเนินการดังกล่าว) ข้อมูลที่เราเก็บรวบรวมจะถูกส่งและเก็บรักษาไว้ในสหรัฐอเมริกาและประเทศอื่นๆ ตามที่ระบุไว้ในคำชี้แจงสิทธิ์ส่วนบุคคลของไมโครซอฟท์

ตำแหน่งที่ตั้ง

บริการตำแหน่งที่ตั้งของไมโครซอฟท์สามารถระบุข้อมูลตำแหน่งที่ตั้งให้แก่อุปกรณ์ Windows โดยใช้การผสมผสานกันระหว่างบริการกำหนดตำแหน่งบนโลก (GPS) จุดเข้าใช้งานแบบไร้สาย เสาเครือข่ายโทรศัพท์ในบริเวณใกล้เคียง และที่อยู่ IP ของคุณ ทั้งนี้จะขึ้นอยู่กับความสามารถของอุปกรณ์ที่คุณใช้งานด้วย

การเปิดใช้งานตำแหน่งที่ตั้งจะทำให้แอป บริการ และคุณสมบัติของ Windows บางอย่างอาจขออนุญาตเข้าถึงและใช้งานข้อมูลตำแหน่งที่ตั้งของคุณเพื่อมอบบริการที่เกี่ยวข้องกับตำแหน่งที่ตั้งได้อย่างแม่นยำตามที่อุปกรณ์ของคุณรองรับ เมื่อแอปหรือบริการที่เกี่ยวข้องกับสถานที่ใช้งานตำแหน่งของคุณ จะมีการเก็บข้อมูลตำแหน่งที่ตั้งของคุณและประวัติตำแหน่งที่ตั้งล่าสุดไว้บนอุปกรณ์ของคุณและส่งให้ไมโครซอฟท์ในรูปแบบที่ไม่ระบุตัวตนเพื่อปรับปรุงบริการค้นหาตำแหน่ง

นอกจากนั้น หากคุณลงชื่อเข้าใช้บัญชีไมโครซอฟท์ข้อมูลตำแหน่งที่ตั้งที่สมบูรณ์ล่าสุดของคุณจะถูกบันทึกไว้ในระบบคลาวด์และเข้าถึงได้จากแอปหรือบริการอื่นๆ โดยใช้บัญชีไมโครซอฟท์ในอุปกรณ์เครื่องใดก็ได้ของคุณ หากอุปกรณ์ของคุณไม่สามารถหาตำแหน่งที่ตั้งที่สมบูรณ์ได้เอง (เช่น อยู่ในอาคารหรือชั้นใต้ดิน) มันจะสามารถใช้ตำแหน่งที่ตั้งที่สมบูรณ์ล่าสุดของคุณที่บันทึกไว้ในระบบคลาวด์ได้

คุณสามารถปิดการเข้าถึงและลบประวัติตำแหน่งที่ตั้งของอุปกรณ์ของคุณได้ทุกเมื่อใน เริ่มต้น > การตั้งค่า > ความเป็นส่วนตัว > ตำแหน่งที่ตั้ง

หากคุณมีอุปกรณ์แบบพกพา เช่น แล็ปท็อป การเปิดตำแหน่งที่ตั้งจะทำให้สามารถใช้งานคุณสมบัติ ค้นหาอุปกรณ์ของฉัน ได้เช่นกัน ซึ่งจะใช้ข้อมูลตำแหน่งที่ตั้งของคุณในการค้นหาอุปกรณ์หากคุณเกิดทำหายขึ้นมา คุณต้องลงชื่อเข้าใช้ Windows ด้วยบัญชีไมโครซอฟท์ของคุณเพื่อให้คุณสมบัตินี้ทำงาน คุณสามารถปิดการใช้งานคุณสมบัตินี้ได้ทุกเมื่อใน เริ่มต้น > การตั้งค่า > การปรับปรุงและความปลอดภัย > ค้นหาอุปกรณ์ของฉัน

การรู้จำเสียง

Windows มีคุณสมบัติการรู้จำเสียงทั้งแบบติดตั้งในอุปกรณ์ (ใช้งานได้ผ่านแอปเดสก์ท็อปการรู้จำเสียงของ Windows) และบริการรู้จำเสียงผ่านระบบคลาวด์ที่เปิดตัวมาพร้อม Cortana ในตลาดและภูมิภาคเดียวกันกับที่ Cortana ให้บริการ หากต้องการทราบว่าตอนนี้รองรับคำพูดภาษาอะไรและภูมิภาคไหนบ้างแล้ว ให้ค้นคำว่า "ภูมิภาคและภาษาของ Cortana" ในเบราว์เซอร์ใดก็ได้ หรือในแถบค้นหา การเปิดการตั้งค่าการรู้จำเสียงทำให้ไมโครซอฟท์สามารถเก็บรวบรวมและใช้งานการบันทึกเสียงของคุณเพื่อให้สามารถจัดหาบริการการรู้จำเสียงระบบคลาวด์ใน Cortana แอปจาก Store ที่รองรับ และในส่วนอื่นๆ ของ Windows เมื่อเวลาผ่านไป นอกจากนั้น เรายังรวบรวมข้อมูลจากพจนานุกรมผู้ใช้ที่สร้างขึ้นในอุปกรณ์ของคุณในฐานะส่วนหนึ่งของบริการดังกล่าว พจนานุกรมผู้ใช้นี้จะเก็บรักษาคำที่มีลักษณะเฉพาะ เช่น ชื่อที่คุณเขียน ซึ่งจะใช้ให้คุณพิมพ์และลงหมึกได้อย่างแม่นยำมากยิ่งขึ้น ทั้งข้อมูลเสียงและพจนานุกรมผู้ใช้จะถูกนำมาใช้ร่วมกันเพื่อช่วยปรับปรุงความสามารถของเราในการรู้จำคำพูดของผู้ใช้ทุกคนได้อย่างถูกต้อง คุณสามารถปิดการใช้งานคุณสมบัตินี้ได้ทุกเมื่อใน เริ่มต้น > การตั้งค่า > ความเป็นส่วนตัว > สั่งงานด้วยเสียง การใช้หมึก และการพิมพ์ หากภายหลังคุณอนุญาตให้ Cortana เข้าถึงข้อมูลต่างๆ เช่น ข้อมูลปฏิทินและรายชื่อติดต่อ ได้ ประสบการณ์การรู้จำเสียงของคุณก็สามารถปรับแต่งให้ใช้งานกับคุณได้ดียิ่งขึ้นได้ โปรดทราบว่าคุณสามารถใช้งานการรู้จำเสียงของอุปกรณ์ได้ตลอดเวลาด้วยการพิมพ์คำว่า "การรู้จำเสียงของ Windows" ในแถบค้นหาและเปิดใช้งานแอปเดสก์ท็อปการรู้จำเสียงของ Windows

การวินิจฉัย

เมื่อคุณใช้งาน Windows เราจะเก็บรวบรวมข้อมูลการวินิจฉัยที่ช่วยให้เราสามารถระบุและแก้ไขปัญหาต่างๆ ได้ รวมถึงช่วยให้อุปกรณ์ของคุณทันสมัยและปลอดภัยอยู่เสมอ นอกจากนั้น การวินิจฉัยยังช่วยให้เราสามารถปรับปรุงผลิตภัณฑ์และการบริการของเรา และมอบประสบการณ์ที่ปรับแต่งแบบส่วนบุคคลให้แก่คุณ โดยจะขึ้นอยู่กับการตั้งค่าที่คุณเลือก ข้อมูลนี้จะถูกส่งให้กับไมโครซอฟท์และจัดเก็บด้วยรหัสเฉพาะหนึ่งชุดหรือมากกว่าหนึ่งชุดขึ้นไปที่สามารถช่วยเราให้จดจำผู้ใช้แต่ละคนบนอุปกรณ์แต่ละอย่างได้ และทำให้เข้าใจปัญหาด้านการบริการและรูปแบบการใช้งานของอุปกรณ์ ข้อมูลการวินิจฉัยมีสองระดับ ได้แก่ แบบเต็ม และ แบบพื้นฐาน

พื้นฐาน ข้อมูลพื้นฐานเป็นข้อมูลที่สำคัญต่อการปฏิบัติการของ Windows ข้อมูลนี้ช่วยให้เรารับรู้ถึงความสามารถของอุปกรณ์ของคุณ มีอะไรที่ติดตั้งแล้วบ้าง และ Windows กำลังทำงานอย่างถูกต้องหรือไม่ เพื่อช่วยให้ Windows และแอปต่างๆ ปลอดภัย ทันสมัย และสามารถทำงานได้อย่างถูกต้อง นอกจากนั้น ตัวเลือกนี้จะเปิดการรายงานข้อผิดพลาดพื้นฐานเพื่อส่งกลับไปยังไมโครซอฟท์ด้วย

ข้อมูลแบบเต็ม หมายถึงทุกสิ่งทุกอย่างที่มากับข้อมูลแบบพื้นฐาน รวมถึงข้อมูลเกี่ยวกับการใช้งานแอป การใช้งานเบราว์เซอร์ ลักษณะและระยะเวลาในการใช้งานคุณสมบัติและแอปบางอย่าง แอปและคุณสมบัติที่คุณใช้บ่อยที่สุด ความถี่ในการใช้งานวิธีใช้และการสนับสนุนของ Windows และบริการที่คุณใช้เพื่อลงชื่อเข้าใช้แอป สำหรับข้อมูลแบบเต็ม ไมโครซอฟท์จะเก็บรวบรวมข้อมูลการวินิจฉัยเพิ่มเติมรวมทั้งสถานะของหน่วยความจำในอุปกรณ์ของคุณตอนที่ระบบหรือแอปหยุดทำงาน (ซึ่งอาจรวมเอาบางส่วนของไฟล์ที่คุณกำลังทำงานอยู่เข้าไปด้วยโดยไม่ได้ตั้งใจเมื่อมีปัญหาเกิดขึ้น) ข้อมูลที่ระบุตัวตนใดๆ จะถูกลบออกจากการพิมพ์และการเขียนของคุณ เช่น คำหรือตัวอักษร เราจะใช้ข้อมูลเพิ่มเติมทั้งหมดนี้ร่วมกันเพื่อปรับปรุงผลิตภัณฑ์และบริการของไมโครซอฟท์เพื่อผู้ใช้ทุกคน

สำหรับข้อมูลทั้งสองระดับแล้ว เราจะไม่ใช้งานข้อมูลนี้เพื่อปรับแต่งผลิตภัณฑ์หรือบริการของไมโครซอฟท์นอกจากจะได้รับอนุญาตจากคุณด้วยการตั้งค่าประสบการณ์เฉพาะตัวที่แยกต่างหากพร้อมข้อมูลวินิจฉัย (ตามที่อธิบายไว้ด้านล่างนี้)

คุณสามารถปรับระดับการเก็บข้อมูลวินิจฉัยของตัวเองได้ทุกเมื่อใน เริ่มต้น > การตั้งค่า > ความเป็นส่วนตัว > คำติชมและการวินิจฉัย

ประสบการณ์การใช้งานที่ปรับให้เหมาะสมด้วยข้อมูลการวินิจฉัย

หากคุณเลือกที่จะเปิดประสบการณ์การใช้งานที่ปรับให้เหมาะสม เราจะใช้งานข้อมูลการใช้งานบางส่วนตามที่ได้อธิบายไว้ข้างต้นในการวินิจฉัยเพื่อปรับแต่งประสบการณ์การใช้งาน Windows รวมถึงผลิตภัณฑ์และการบริการอื่นๆ ของคุณ ในส่วนนี้จะรวมถึงการแนะนำวิธีกำหนดและปรับแต่ง Windows และคำแนะนำและข้อเสนอของคุณสมบัติต่างๆ ของ Windows ตลอดจนแอป บริการ ฮาร์ดแวร์ และอุปกรณ์ต่อพ่วงที่รองรับ ตัวอย่างเช่น เพื่อช่วยให้คุณใช้งานอุปกรณ์ของตัวเองได้อย่างเต็มศักยภาพ เราอาจบอกคุณถึงคุณสมบัติที่คุณอาจยังไม่รู้จักหรือที่เพิ่งออกมาใหม่ หากคุณประสบปัญหากับอุปกรณ์ Windows เราก็อาจเสนอวิธีแก้ปัญหาให้คุณ คุณอาจได้รับข้อเสนอในการกำหนดหน้าจอล็อกด้วยรูปภาพ หรือได้เห็นภาพประเภทที่คุณชอบหรือไม่ชอบมากขึ้นหรือน้อยลงได้ หากคุณกำลังสตรีมดูภาพยนตร์ในเบราว์เซอร์อยู่ ก็อาจมีการแนะนำแอปจาก Store ที่สตรีมได้มีประสิทธิภาพมากยิ่งขึ้นได้ หรือถ้าหน่วยความจำในฮาร์ดไดร์ฟของคุณใกล้จะเต็ม Windows ก็อาจแนะนำให้คุณลองใช้บริการพื้นที่จัดเก็บแบบออนไลน์ OneDrive หรือซื้อพื้นที่เพิ่มขึ้นถ้าจำเป็น ขอบเขตของข้อมูลการใช้งานจะขึ้นอยู่กับว่าคุณเลือกการรายงานการวินิจฉัยแบบเต็มหรือแบบพื้นฐาน ตัวอย่างเช่น ถ้าคุณเลือกแบบเต็ม ข้อมูลของคุณจะรวมถึงการใช้งานเบราว์เซอร์และแอปด้วย อย่างไรก็ตาม ถึงแม้คุณจะเปิดการใช้งานประสบการณ์การใช้งานที่ปรับให้เหมาะสม เราก็จะไม่นำข้อมูลความเสียหาย คำพูด การพิมพ์ หรือการใช้หมึกมาใช้เพื่อการตั้งค่าส่วนบุคคล คุณสามารถปิดการใช้งานคุณสมบัตินี้ได้ทุกเมื่อใน เริ่มต้น > การตั้งค่า > ความเป็นส่วนตัว > คำติชมและการวินิจฉัย

โฆษณาที่เกี่ยวข้อง

Windows สร้างรหัสโฆษณาที่ไม่ซ้ำกันสำหรับผู้ใช้แต่ละรายในอุปกรณ์ ซึ่งผู้พัฒนาแอปพลิเคชันและเครือข่ายการโฆษณาสามารถนำรหัสนี้ไปใช้เพื่อแสดงโฆษณาที่เกี่ยวข้องมากขึ้นในแอป เมื่อเปิดใช้งานรหัสโฆษณา แอปจะสามารถเข้าถึงและใช้รหัสโฆษณาที่ค่อนข้างคล้ายกันมากกับที่เว็บไซต์ต่างๆ เข้าถึงและใช้รหัสเฉพาะที่จัดเก็บไว้ในคุกกี้ ดังนั้น รหัสโฆษณาของคุณสามารถใช้งานได้โดยนักพัฒนาแอป (และเครือข่ายการโฆษณาที่พวกเขาทำงานด้วย) เพื่อมอบการโฆษณาที่เกี่ยวข้องและประสบการณ์การใช้งานที่ปรับให้เป็นส่วนตัวในแอปของผู้พัฒนาเหล่านั้นทุกแอป คุณสามารถปิดคุณสมบัตินี้ได้ทุกเมื่อใน เริ่มต้น > การตั้งค่า > ความเป็นส่วนตัว โปรดทราบว่าการปิดรหัสโฆษณาจะไม่ลดจำนวนโฆษณาที่คุณจะได้เห็นลงไป แต่อาจหมายความว่าโฆษณาจะน่าสนใจและเกี่ยวข้องกันคุณน้อยลง การเปิดใช้งานอีกครั้งจะเริ่มต้นรหัสโฆษณาของคุณใหม่

Windows Defender SmartScreen

Windows Defender SmartScreen จะส่งข้อมูลเกี่ยวกับเว็บไซต์ที่คุณเข้าชมและไฟล์ที่คุณดาวน์โหลดมาที่ไมโครซอฟท์เพื่อเตือนและช่วยปกป้องคุณและอุปกรณ์ของคุณจากเนื้อหาเว็บที่ไม่ปลอดภัยหรือซอฟต์แวร์ที่เป็นอันตราย เนื่องจากเราพยายามอย่างยิ่งที่จะปกป้องคุณเวลาที่ใช้บริการของเราและของบุคคลภายนอก เราจึงเปิดการใช้งาน Windows Defender SmartScreen เป็นค่าเริ่มต้น คุณสามารถปิดการใช้งาน Windows Defender SmartScreen ได้ทุกเมื่อใน เริ่มต้น > การตั้งค่า > ความเป็นส่วนตัว

รีวิวการตั้งค่าความเป็นส่วนตัวของคุณเพื่อเตรียมพร้อมสำหรับการอัปเดตถัดไป

เพียงแค่ไม่กี่นาที จากนั้นคุณสามารถกลับไปทำสิ่งที่ทำอยู่ได้

  • การอัปเดต Windows 10 สำหรับนักสร้างสรรค์กำลังจะมาในเร็วๆ นี้ เราต้องการตรวจสอบการตั้งค่าความเป็นส่วนตัวของคุณก่อนที่การอัปเดตถัดไปจะมาถึง

  • เราจะบันทึกการตั้งค่าของคุณและแจ้งให้คุณทราบก่อนการอัปเดตจะเริ่มต้นขึ้น การอัปเดตจะไม่เกิดขึ้นทันที

เรากำลังเตรียมการอัปเดตสำหรับอุปกรณ์ของคุณ ระบบจะดาวน์โหลดในพื้นหลัง ดังนั้นคุณสามารถทำงานต่อไปได้ ระบบจะติดตั้งการอัปเดตตามการตั้งค่า Windows Update

ไมโครซอฟท์ให้คุณควบคุมความเป็นส่วนตัวของคุณเอง คำแนะนำต่อไปนี้เป็นไปตามการตั้งค่าปัจจุบันของคุณ การเปลี่ยนแปลงใดๆ ที่คุณทำขณะนี้จะมีผลกับบัญชีทั้งหมดบนอุปกรณ์นี้เมื่อคุณติดตั้งการอัปเดต Windows 10 สำหรับนักสร้างสรรค์ ค้นหาเพื่อดูการตั้งค่าทั้งหมด

ตำแหน่งที่ตั้ง

รับประสบการณ์การใช้งานตามตำแหน่งที่ตั้ง เช่น สภาพอากาศในท้องถิ่นและเส้นทางไปยังสถานที่ที่คุณชื่นชอบ ให้ Windows และแอปขอตำแหน่งที่ตั้งของคุณ และส่งข้อมูลตำแหน่งที่ตั้งให้ Microsoft เพื่อปรับปรุงบริการตำแหน่งที่ตั้ง

Windows และแอปไม่สามารถใช้ตำแหน่งที่ตั้งของคุณในการให้ข้อมูล เช่น สภาพอากาศในท้องถิ่น เส้นทาง และค้นหาอุปกรณ์ของฉัน

ในระดับพื้นฐาน คุณจะส่งปริมาณข้อมูลน้อยกว่าให้ไมโครซอฟท์ เพื่อช่วยแก้ไขข้อผิดพลาดที่คุณพบ

เคล็ดลับที่คุณได้จะเป็นเรื่องทั่วไปมากขึ้นและคำแนะนำอาจเกี่ยวข้องกับคุณน้อยลง

ใช้เวลาไม่นาน แต่ถ้าคุณไม่พร้อมตรวจสอบในตอนนี้ เราก็สามารถ เลื่อนได้อีกหนึ่งครั้ง

  • เราอยากให้คุณติดตั้งการอัปเดตล่าสุดเพื่อให้ Windows ทำงานได้อย่างราบรื่น

  • เราจะบันทึกการตั้งค่าของคุณและแจ้งให้คุณทราบก่อนการอัปเดตจะเริ่มต้นขึ้น การอัปเดตจะไม่เกิดขึ้นทันที

ถึงเวลาตรวจสอบการตั้งค่าความเป็นส่วนตัวของคุณแล้วเพื่อเตรียมพร้อมสำหรับการอัปเดตถัดไป

ใช้เวลาไม่นาน เราได้แสดงตัวเตือนนี้มากกว่า 5 ครั้งแล้ว ตอนนี้เราขอให้คุณช่วยอัปเดตตามที่เตรียมพร้อมไว้แล้วสำหรับพีซีของคุณ

  • เราอยากให้คุณติดตั้งการอัปเดตล่าสุดเพื่อให้ Windows ทำงานได้อย่างราบรื่น

  • เราจะบันทึกการตั้งค่าของคุณและแจ้งให้คุณทราบก่อนการอัปเดตจะเริ่มต้นขึ้น การอัปเดตจะไม่เกิดขึ้นทันที

เลือก ตรวจสอบการตั้งค่า แล้วเราจะดำเนินการให้เอง
