簡介

當您設定 Windows 時，我們會要求您選擇與您的隱私權相關的設定。 我們會在設定畫面上建議我們認為適當的設定，讓您的裝置在連接服務時更可靠和安全，為您提供更為個人化的體驗並協助改善我們的服務。 您現在可以選擇您的設定，或者像往常一樣隨時移至 [開始] > [設定] 更新這些設定。

以下資訊說明我們將根據您選擇的設定收集哪些資料與資料的使用方式。 請務必檢視完整的 Microsoft 隱私權聲明，以深入了解我們在您使用 Windows 時收集的所有個人資料與使用方式 (在任何瀏覽器視窗中輸入 aka.ms/privacy 即可得知)。 如 Microsoft 隱私權聲明中所述，我們收集的資料是傳送到並儲存在美國和其他國家/地區。

位置

Microsoft 的位置服務結合使用全球定位服務 (GPS)、鄰近的無線存取點、基地台和 IP 位址 (取決於裝置的功能) 來提供 Windows 裝置的位置資訊。

開啟 [位置] 可讓特定應用程式、服務及 Windows 功能要求權限存取和使用您的位置資料，以您裝置所支援的精確層級提供定位感知服務。 當定位感知應用程式或服務使用您的位置時，您的位置資訊及最近的位置歷程記錄會儲存在您的裝置上，並以去除識別資訊的格式傳送至 Microsoft 以改善位置服務。

此外，如果您使用您的 Microsoft 帳戶登入，最後一個已知的準確位置資訊會儲存到雲端，並可供在您的所有裝置間使用您的 Microsoft 帳戶的其他應用程式或服務使用。 如果您的裝置無法自行取得良好的位置 (例如在建築物或地下室內)，它可以使用儲存在雲端中您的最後一個已知良好的位置。

您隨時可以在 [開始] > [設定] > [隱私權] > [位置] 中關閉位置存取功能，以及清除您裝置的位置歷程記錄。

如果您有可攜式裝置，例如膝上型電腦，開啟位置也會啟用「尋找我的裝置」功能，使用您的位置資料來協助您尋找您的裝置 (若遺失的話)。 若要使用這項功能，您必須使用您的 Microsoft 帳戶登入 Windows。 您隨時可以在 [開始] > [設定] > [更新與安全性] > [尋找我的裝置] 關閉此功能。

語音辨識

Windows 提供以裝置為準的語音辨識功能 (透過 Windows 語音辨識傳統型應用程式) 和以雲端為準的語音辨識服務 (在提供 Cortana 的市場和地區隨 Cortana 一起推出) 兩種。 若要了解目前支援哪些語言和地區語音，請在任何瀏覽器或搜尋列中搜尋「Cortana 的地區和語言」。 開啟 [語音辨識] 設定可讓 Microsoft 收集和使用您的錄音，在 Cortana、支援的市集應用程式以及隨著時間在 Windows 的其他部分中提供您以雲端為準的語音辨識服務。 做為該服務的一部分，我們還會從您裝置上建立的使用者字典收集資訊。 這個使用者字典儲存獨特的字詞，例如您寫入的名稱，可協助您更精準地輸入和手寫筆跡。 語音資料和使用者字典二者都用於彙總，以協助提升我們正確辨識所有使用者語音的能力。 您隨時可以在 [開始] > [設定] > [隱私權] > [語音、手寫筆跡，以及鍵入] 關閉此功能。 如果您稍後允許 Cortana 存取資訊，例如行事曆和連絡人資料，可以將您的語音辨識體驗個人化，使其更適合您使用。 請注意，您一直都可以在搜尋列中輸入「Windows 語音辨識」並執行 Windows 語音辨識傳統型應用程式，來使用以裝置為準的語音辨識功能。

診斷

當您使用 Windows 時，我們會收集資料，以協助我們找出並進行問題的疑難排解，確保您的裝置在最新狀態並且安全無虞。 根據您選擇的設定，診斷也可協助我們改善我們的產品與服務，並提供您個人化的體驗。 此資料會傳送至 Microsoft 並使用一個或多個唯一識別碼加以儲存，唯一識別碼可協助我們識別個別裝置上的個別使用者，以及了解裝置的服務問題和使用模式。 診斷資料有兩種層級︰完整和基本。

基本 資料是對於 Windows 運作非常重要的資料。 該資料可讓我們知道您裝置的功能、已安裝的項目，以及 Windows 是否正常運作，以協助我們保護 Windows 和應用程式的安全、維持在最新狀態和正常執行。 此選項也包括傳回基本錯誤報告給 Microsoft。

完整 資料包括 [基本] 層級資料收集的所有項目，再加上有關應用程式使用量、瀏覽器使用量、您如何使用某些功能或應用程式及使用時間長短、哪些應用程式與功能您最常使用、您多常使用 Windows 說明及支援，以及您用來登入應用程式的服務等資料。 Microsoft 會在 [完整] 層級收集額外的診斷資料，包括當系統或應用程式發生損毀時裝置的記憶體狀態 (這可能會在無意中包含問題發生時您正在使用的檔案的部分內容)。 所有識別資訊將從您鍵入和手寫的輸入，例如字詞或字元中移除。 我們會將所有這個額外資料用於彙總，為所有使用者改善 Microsoft 的產品和服務。

對於這兩個層級，我們不會使用此資料來個人化 Microsoft 的產品或服務，除非您讓我們這麼做，利用診斷資料 (如下所述) 提供量身打造的個別設定。

您隨時可以在 [開始] > [設定] > [隱私權] > [意見反應與診斷] 中調整您的診斷資料收集層級。

利用診斷資料量身打造體驗

如果您選擇獲得量身打造的體驗，我們會使用上述「診斷」中的一些使用方式資料來個人化您的 Windows 及其他產品和服務的體驗。 這包括建議如何自訂和最佳化 Windows；以及 Windows 功能與支援的應用程式、服務、硬體和周邊裝置的建議與優惠。 例如，協助您充分運用您的裝置，我們會告訴您可能不知道或全新的功能。 如果您有 Windows 裝置方面的問題，我們會提供您解決方案。 您還可以利用圖片自訂您的鎖定畫面，或顯示更多您喜歡的圖片類型或較少您不喜歡的圖片。 如果您在瀏覽器中串流電影，可能會建議您市集中可更有效率進行串流處理的應用程式。 或者，如果您的硬碟空間不足，Windows 可能會建議您嘗試 OneDrive 線上儲存空間服務，或者您若需要的話，建議您購買更多空間。 使用方式資料的範圍將視您選擇 [完整] 或 [基本診斷] 報告而定。 例如，如果您選擇 [完整]，您的資料會包含瀏覽器與應用程式使用方式。 不過，即使您啟用量身打造的體驗，我們並不會使用您的當機、語音、鍵入或筆跡輸入資料進行個人化。 您隨時可以在 [開始] > [設定] > [隱私權] > [意見反應與診斷] 關閉此項功能。

相關廣告

Windows 會在裝置上針對每個使用者產生一組唯一的廣告識別碼，應用程式開發人員和廣告網路將可以使用這組識別碼在應用程式中提供更相關的廣告。 當廣告識別碼啟用時，應用程式就能夠存取及使用廣告識別碼，使用方式和網站存取及使用 Cookie 中儲存之唯一識別碼的方式相同。 因此，應用程式開發人員 (以及他們合作的廣告網路) 就可以使用您的廣告識別碼，在他們的應用程式之間提供更相關的廣告和其他個人化體驗。 您隨時可以在 [開始] > [設定] > [隱私權] 關閉此項功能。 請注意，關閉廣告識別碼並不會減少您所看到廣告的數目，但它可能表示廣告比較不吸引人以及與您相關。 再次開啟會重設廣告識別碼。
