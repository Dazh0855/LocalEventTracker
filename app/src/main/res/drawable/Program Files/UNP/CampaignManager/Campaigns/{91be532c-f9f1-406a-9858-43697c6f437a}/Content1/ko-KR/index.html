Windows 10에 새 기능 출시 예정

Windows 10의 가장 큰 장점 중 하나는 지속적으로 개선되고 있다는 점입니다. 새 기능이 추가되고, 보안이 업데이트되며, 최신 상태로 유지하는 방법도 간단합니다. 자동 업데이트를 사용하도록 설정하면 현재 사용 중인 디바이스의 수명 주기 동안 새로운 Windows 10 기능과 향상된 보안이 출시되는 즉시 무료로 제공됩니다.*

곧 출시될 차세대 혁신은 Windows 10 크리에이터 업데이트입니다. 이 업데이트는 향상된 최신 보안 그리고 다음과 같은 다양하고 멋진 새 기능을 제공합니다.

  • Windows 10의 3D - 완전히 새로운 차원에서 만들고 공유할 수 있는 새로운 그림판 3D 응용 프로그램 탑재
  • PC 게임 - 4K 게임, 게임 모드 및 기본 제공 게임 브로드캐스트
  • 보안 대시보드 디스플레이 및 디바이스 상태 관리자

Windows 10 환경은 계속해서 개선되고 있습니다!

일부 기능은 하드웨어에 따라 다릅니다. * 인터넷 액세스 요금이 부과될 수 있습니다.

소개

Windows를 설정할 때 사용자의 개인 정보와 관련된 설정을 선택하라는 메시지가 표시됩니다. 설정 화면에서는 서비스에 연결하는 동안 디바이스의 신뢰도와 보안을 높이고 보다 개인 설정된 환경을 제공하고 Microsoft 서비스를 개선하는 데 도움이 되는 적절한 설정이 제공됩니다. 이제 설정을 선택할 수도 있고, 평소처럼 언제든지 시작 > 설정으로 이동하여 설정을 업데이트할 수도 있습니다.

아래는 사용자가 선택하는 설정에 따라 어떤 데이터가 수집되고 어떤 방식으로 사용되는지 설명하는 정보입니다. Windows를 사용할 때 어떤 개인 데이터가 수집되고 어떤 방식으로 사용되는지에 대한 자세한 내용은 Microsoft 개인정보취급방침을 검토하세요(아무 브라우저 창에 aka.ms/privacy 입력). 수집된 데이터는 Microsoft 개인정보취급방침에 명시된 대로 미국 및 기타 국가로 전송 및 저장됩니다.

위치

Microsoft의 위치 서비스는 사용자의 디바이스 기능에 따라 글로벌 위치확인 서비스(GPS), 가까운 무선 액세스 지점, 셀 타워 및 IP 주소를 조합하여 Windows 디바이스에 위치 정보를 제공합니다.

위치를 켜면 특정 앱, 서비스 및 Windows 기능이 사용자의 위치 데이터에 액세스하여 사용자의 디바이스 지원만큼 정확한 수준에서 위치 인식 서비스를 제공하는 데 필요한 권한을 요청합니다. 위치 인식 앱 또는 서비스에서 사용자의 위치 정보가 사용되면 위치 서비스를 개선하기 위해 사용자의 위치 정보 및 최근 위치 기록이 사용자의 디바이스에 저장되고 개인 식별 정보가 제거된 형식으로 Microsoft에 전송됩니다.

또한 Microsoft 계정으로 로그인하면 마지막으로 알려진 올바른 위치 정보가 클라우드에 저장되어 사용자 디바이스에서 Microsoft 계정을 사용하는 다른 앱 또는 디바이스에 제공됩니다. 사용자 디바이스가 자체적으로 올바른 위치 정보를 얻을 수 없는 경우(예를 들어 건물 내부나 지하실에 있어서) 클라우드에 저장된 마지막으로 알려진 올바른 위치 정보를 사용할 수 있습니다.

언제든 시작 > 설정 > 개인 정보 > 위치에서 위치 액세스를 끄고 디바이스의 위치 기록을 지울 수 있습니다.

노트북 같은 휴대용 장치를 사용하는 경우 위치를 켜면 디바이스를 분실했을 때 위치 데이터를 사용하여 디바이스를 찾을 수 있도록 도와주는 내 디바이스 찾기 기능도 활성화됩니다. 이 기능이 작동하려면 Microsoft 계정으로 Windows에 로그인해야 합니다. 언제든 시작 > 설정 > 업데이트 및 보안 > 네 디바이스 찾기에서 이 기능을 끌 수 있습니다.

음성 인식

Windows는 디바이스 기반 음성 인식 기능(Windows 음성 인식 데스크톱 앱을 통해 제공), 그리고 Cortana가 제공되는 시장 및 지역에서 Cortana와 함께 도입된 클라우드 기반 음성 인식 서비스를 제공합니다. 현재 음성이 지원되는 언어와 지역을 알아보려면 아무 브라우저에서 또는 검색창에서 "Cortana 지원 지역 및 언어"로 검색하세요. 음성 인식 설정을 켜면 Microsoft에서 사용자의 음성 녹음을 사용하여 Cortana, 지원되는 스토어 앱 그리고 향후에는 Windows의 다른 부분에서 사용자에게 클라우드 기반 음성 인식 서비스를 제공할 수 있습니다. 이 서비스의 일환으로, 사용자의 디바이스에 생성된 사용자 사전의 정보도 수집됩니다. 이 사용자 사전에는 사용자가 쓰는 이름 같은 고유의 단어가 저장되며 이러한 정보는 입력 정확도를 높이는 데 도움이 됩니다. 음성 데이터와 사용자 사전은 모든 사용자의 음성을 정확하게 인식하도록 기능을 개선하기 위해 집계에 이용됩니다. 언제든 시작 > 설정 > 개인 정보 > 음성, 수동 입력 및 입력에서 이 기능을 끌 수 있습니다. 나중에 Cortana가 일정 및 연락처 데이터 등의 정보에 액세스할 수 있도록 허용하면 음성 인식 환경을 사용자에게 더 적합하게 개인 설정할 수 있습니다. 언제든지 검색창에 "Windows 음성 인식"을 입력하고 Windows 음성 인식 데스크톱 앱을 실행하여 디바이스 기반 음성 인식을 사용할 수 있습니다.

진단

사용자가 Windows를 사용할 때 문제를 식별하여 해결하고 디바이스를 최신 상태로 안전하게 유지하기 위해 진단 데이터가 수집됩니다. 사용자가 선택하는 설정에 따라 진단은 제품 및 서비스를 개선하고 개인 설정된 환경을 제공하는 데 도움이 될 수 있습니다. 이 데이터는 Microsoft로 전송되어 개별 디바이스의 개별 사용자를 인식하고 디바이스의 서비스 문제 및 사용 패턴을 파악하는 데 도움이 되는 하나 이상의 고유 식별자와 함께 저장됩니다. 진단 데이터의 수준은 전체와 기본의 두 가지가 있습니다.

기본 Windows 운영에 중요한 데이터입니다. 기본 데이터는 디바이스의 기능, 설치된 항목, Windows의 정상 작동 여부를 알려주어 Windows와 앱의 보안을 유지하고, 최신 상태를 유지하고, 정상적으로 실행하는 데 도움을 줍니다. 이 옵션에는 다시 Microsoft로 전송되는 기본 오류 보고가 포함되어 있습니다.

전체 기본 데이터를 통해 수집한 모든 정보와 앱 사용, 브라우저 사용, 특정 기능 또는 앱을 사용하는 방식 및 사용 시간, 가장 많이 사용하는 앱 및 기능, Windows 도움말 및 지원을 사용하는 빈도, 앱에 로그인할 때 사용하는 서비스에 대한 데이터가 포함됩니다. 전체 옵션을 사용하면 Microsoft에서 시스템이나 앱 크래시가 발생할 때의 디바이스 메모리 상태를 포함하여 추가 진단 데이터를 수집합니다(문제가 발생한 당시에 사용자가 사용하던 파일의 일부가 의도치 않게 포함될 수 있음). 신원 파악이 가능한 모든 정보는 사용자가 타이핑과 필기로 적은 입력(예: 단어 또는 문자)에서 제거됩니다. Microsoft에서는 이 모든 추가 데이터를 집계에 사용하여 모든 사용자를 위해 Microsoft 제품 및 서비스를 개선합니다.

어떤 수준을 선택하든, 사용자가 진단 데이터를 이용한 별도의 맞춤형 환경 설정을 통해 허락하지 않는 한(설명은 아래 참조) Microsoft는 이 데이터를 Microsoft 제품 또는 서비스를 개인 설정하는 데 사용하지 않습니다.

언제든 시작 > 설정 > 개인 정보 > 피드백 및 진단에서 진단 데이터 수집 수준을 조정할 수 있습니다.

진단 데이터를 이용한 맞춤형 환경

사용자가 맞춤형 환경을 사용하기로 선택하면 Microsoft에서는 위에 설명된 사용 현황 데이터 중 일부를 진단에 사용하여 Windows와 기타 제품 및 서비스를 통해 사용자의 환경을 개인 설정합니다. 여기에는 Windows를 사용자 지정 및 최적화 하는 방법에 대한 권장 사항, 그리고 Windows 기능과 지원되는 앱, 서비스, 하드웨어 및 주변 장치의 추천 사항 및 제안 사항이 포함됩니다. 예를 들어 사용자가 디바이스를 최대한 활용할 수 있도록 사용자가 잘 모를 것 같은 기능 또는 새로 나온 기능에 대해 사용자에게 알려줍니다. Windows 디바이스에 문제가 발생하는 경우 사용자에게 해결책을 제시해 줄 수 있습니다. 사진을 사용하여 잠금 화면을 사용자 지정하거나, 마음에 드는 사진을 더 표시하거나, 마음에 들지 않는 사진을 표시하지 않을 수 있습니다. 브라우저에서 영화를 스트리밍하는 경우 더 효율적으로 스트리밍하는 앱을 스토어에서 추천 받을 수 있습니다. 또는 하드 드라이브에 공간이 부족한 경우 OneDrive 온라인 저장소 서비스를 사용하거나 필요하다면 추가 공간을 구입하라고 Windows에서 권장할 수 있습니다. 사용 현황 데이터의 범위는 전체 또는 기본 진단 보고 중에 무엇을 선택하는지에 따라 달라집니다. 예를 들어 전체를 선택하면 데이터에 브라우저 및 앱 사용 현황이 포함됩니다. 그러나 맞춤형 환경을 사용하도록 설정하더라도 사용자의 크래시, 음성 입력, 키 입력 또는 수동 입력 데이터를 개인 설정에 사용하지는 않습니다. 언제든 시작 > 설정 > 개인 정보 > 피드백 및 진단에서 이 기능을 끌 수 있습니다.

관련성 있는 광고

Windows는 디바이스의 각 사용자를 위한 고유의 광고 ID를 생성합니다. 애플리케이션 개발자와 광고 네트워크는 이를 사용하여 앱에서 사용자와 더 관련이 있는 광고를 표시할 수 있습니다. 광고 ID가 활성화되면 웹 사이트에서 쿠키에 저장된 고유 식별자에 액세스하여 사용하는 것과 똑같은 방법으로 앱에서 광고 ID에 액세스하여 사용할 수 있습니다. 따라서 개발자(그리고 개발자가 사용하는 광고 네트워크)는 여러분의 광고 ID를 사용하여 앱에서 사용자에게 더 관련이 있는 광고와 기타 개인 설정된 환경을 제공할 수 있습니다. 언제든 시작 > 설정 > 개인 정보에서 이 기능을 끌 수 있습니다. 광고 ID를 꺼도 표시되는 광고의 수는 줄어들지 않지만, 흥미가 없고 관련이 없는 광고가 표시될 수 있음을 유의하십시오. 다시 켜면 광고 ID가 초기화됩니다.

Windows Defender SmartScreen

Windows Defender SmartScreen은 사용자가 방문하는 웹사이트와 다운로드하는 파일에 대한 데이터를 Microsoft에 전송하여 사용자에게 주의를 주고 안전하지 않은 웹 콘텐츠나 악성 소프트웨어로부터 사용자와 사용자의 디바이스를 보호합니다. 사용자가 Microsoft 서비스와 타사 서비스를 사용할 때 사용자를 보호하기 위해 Windows Defender SmartScreen이 기본적으로 켜져 있습니다. 언제든 시작 > 설정 > 개인 정보에서 Windows Defender SmartScreen을 끌 수 있습니다.

현재 디바이스 업데이트를 준비하는 중입니다. 업데이트는 백그라운드에서 다운로드되므로 하던 일을 계속 진행하셔도 됩니다. 업데이트는 사용자의 Windows 업데이트 설정에 따라 설치됩니다.

지금은 업데이트를 설치하기 곤란하면 나중에를 선택하세요. 그러면 저희가 나중에 다시 확인하겠습니다.

나중에

장치의 개인 정보 설정 선택

Microsoft는 사용자 자신이 자신의 개인 정보를 제어하도록 하고 있습니다. 다음 제한 사항은 사용자의 현재 설정에 따라 제안된 것입니다. 지금부터 사용자가 변경하는 내용은 Windows 10 크리에이터 업데이트를 설치할 때 이 장치의 모든 계정에 적용됩니다. 모든 설정을 보려면 스크롤하세요.

켬끔
위치

해당 지역 날씨 및 즐겨 찾는 장소에 대한 길 찾기와 같은 위치 기반 환경을 사용합니다. Windows 및 앱에서 위치를 요청하도록 허용하고 Microsoft에 위치 데이터를 전송하여 위치 서비스를 개선하도록 돕습니다.

Windows 및 앱에서 사용자의 위치 정보를 사용하여 해당 지역 날씨, 길 찾기 및 내 장치 찾기 등과 같은 기능을 제공할 수 없습니다.

켬끔
음성 인식

음성 인식을 지원하는 Cortana 및 스토어 앱에 말하세요. Microsoft에 음성 입력을 보내면 음성 명령 서비스를 개선하는 데 도움이 됩니다.

Cortana 또는 스토어 앱을 통한 음성 인식은 지원되지 않습니다.

전체기본
진단

오류를 해결하고 Microsoft 제품 및 서비스를 개선할 수 있도록 도와주세요. 진단 데이터(브라우저, 앱 및 기능 사용 현황, 수동 입력 및 타이핑 데이터 등)를 Microsoft에 보내주시면 됩니다.

기본 수준에서는, 발생한 오류를 해결하는 데 도움이 되는 적은 양의 데이터를 Microsoft에 보냅니다.

켬끔
진단 데이터를 이용한 맞춤식 환경

사용자의 요구에 맞는 Microsoft 제품 및 서비스에 대한 보다 관련성 있는 팁 및 추천 사항이 표시됩니다. Microsoft가 진단 데이터를 사용하여 이러한 동작을 실행하도록 허용합니다.

보다 일반적인 팁이 표시되며, 추천 사항의 관련성이 떨어질 수 있습니다.

켬끔
관련성 있는 광고

앱에서 광고 ID를 사용하여 사용자의 앱 사용도를 기반으로 맞춤형 광고를 표시하도록 허용합니다.

표시되는 광고 수는 변하지 않지만 관련성이 떨어질 수 있습니다.

위 설정, Windows Defender SmartScreen 작동 방식, 관련 데이터 전송 및 사용에 대한 자세한 내용을 보려면 자세한 정보를 선택하세요.
