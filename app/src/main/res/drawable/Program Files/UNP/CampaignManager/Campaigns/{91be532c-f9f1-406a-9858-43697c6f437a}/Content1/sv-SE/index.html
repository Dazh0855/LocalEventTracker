Intro

När du konfigurerar Windows ber vi dig att välja inställningar som rör din integritet. På installationsskärmen rekommenderar vi de inställningar vi anser lämpliga för att göra enheten mer tillförlitlig och säker när du ansluter till tjänster, för att ge dig en mer personlig upplevelse samt för att förbättra Microsofts tjänster. Du kan välja inställningar nu eller så kan du, som vanligt, uppdatera dem när du vill genom att gå till Start > Inställningar.

Informationen nedan förklarar vilken information vi samlar in och hur den används, beroende på vilka inställningar du väljer. Läs igenom Microsofts fullständiga sekretesspolicy om du vill ha mer information om alla de personliga uppgifter vi samlar in och hur de används när du använder Windows (skriv bara aka.ms/privacy i valfri webbläsare). De uppgifter vi samlar in skickas till och lagras i USA och andra länder, vilket beskrivs i Microsofts sekretesspolicy.

Positionering

Microsofts positioneringstjänst fastställer både en Windows-enhets exakta position och dess ungefärliga position med hjälp av GPS, närliggande trådlösa åtkomstpunkter, mobilmaster och din IP-adress, beroende på enhetens funktioner.

Om du aktiverar Position kan vissa appar, tjänster och Windows-funktioner be om tillstånd att få åtkomst till och använda din positionsinformation för att tillhandahålla platsberoende tjänster som din enhet stöder. När din position används av en platsberoende app eller tjänst lagras din positionsinformation och den senaste positionshistoriken på enheten och skickas till Microsoft i ett avidentifierat format för att förbättra positioneringstjänsterna.

Om du har loggat in med ditt Microsoft-konto är dessutom din senast kända positionsinformation sparad i molnet och den är tillgänglig för andra appar och tjänster på alla dina enheter genom ditt Microsoft-konto. Om enheten inte kan hämta en bra position själv (till exempel i en byggnad eller källare), kan den använda din senast kända position som lagrats i molnet.

Du kan stänga av positionsåtkomsten och rensa enhetens positionshistorik när du vill via Start > Inställningar > Sekretess > Positionering.

Om du har en bärbar enhet, till exempel en bärbar dator, får du också tillgång till funktionen Hitta min enhet om du aktiverar positionering. Funktionen använder din positionsinformation för att hjälpa dig hitta enheten om du blir av med den. För att den här funktionen ska fungera måste du logga in på Windows med ditt Microsoft-konto. Du kan inaktivera den när du vill via Start > Inställningar > Uppdatering och säkerhet > Hitta min enhet.

Taligenkänning

Windows erbjuder både en enhetsbaserad taligenkänningsfunktion (tillgänglig via skrivbordsappen Windows Taligenkänning) och en molnbaserad tjänst för taligenkänning. Den senare presenterades tillsammans med Cortana på de marknader och i de regioner där Cortana är tillgänglig. Vill du få mer information om vilka språk och regioner som stöder tal för närvarande, söker du efter ”Cortanas regioner och språk” i en webbläsare eller i sökfältet. Om du aktiverar Taligenkänning kan Microsoft samla in och använda dina röstinspelningar för att ge dig molnbaserade taligenkänningstjänster i Cortana, Store-appar som stöds och, i framtiden, även i andra delar av Windows. Som en del av den här tjänsten samlar vi också in information från den egna ordlistan som skapats på enheten. I den här användarordlistan lagras unika ord, till exempel namn som du skriver, vilket hjälper dig att ange och skriva rätt. Både röstdata och den egna ordlistan används tillsammans för att förbättra vår möjlighet att känna igen talkommandon för alla användare på rätt sätt. Du kan inaktivera detta när du vill via Start > Inställningar > Sekretess > Tal, skrift och inmatning. Om du senare ger Cortana tillgång till information, som kalender och kontaktuppgifter, kan taligenkänningsupplevelsen anpassas så att den passar dig bättre. Observera att du alltid kan använda enhetsbaserad taligenkänning genom att skriva ”Windows Taligenkänning” i sökfältet och köra skrivbordsappen Windows Taligenkänning.

Diagnostik

När du använder Windows samlar vi in diagnostikdata som hjälper oss att identifiera och felsöka problem och hålla enheterna uppdaterade och säkra. Beroende på vilka inställningar du väljer kan också diagnostiken hjälpa oss att förbättra våra produkter och tjänster och ge dig mer anpassade användningsupplevelser. Dessa data skickas till Microsoft och lagras med en eller flera unika identifierare som kan hjälpa oss att känna igen en enskild användare på en viss enhet och förstå enhetens tjänstproblem och användningsmönster. Det finns två nivåer för diagnostikdata: Fullständig och Grundläggande.

Grundläggande data är data som är nödvändiga för att Windows ska fungera. Den gör att vi får veta vilka funktioner som finns på enheten, vad som är installerat och om Windows fungerar korrekt. Den hjälper oss också att hålla Windows och appar säkra, uppdaterade och fungerande. Det här alternativet innefattar grundläggande felrapportering tillbaka till Microsoft.

Fullständiga data omfattar allt som samlas in via Grundläggande data samt information om appanvändning, användning av webbläsaren, hur du använder vissa funktioner eller appar och hur länge, vilka appar och funktioner du använder mest, hur ofta du använder Windows Hjälp och support och vilka tjänster du använder för att logga in på appar. Med alternativet Fullständig samlar Microsoft in ytterligare diagnostikdata som omfattar enhetens minnesstatus när systemet eller en app kraschar (dessa data kan oavsiktligt innehålla delar av en fil som du använde när problemet uppstod). All information som gör att du kan identifieras tas bort från din angivna eller handskrivna inmatning, till exempel ord och tecken. Vi använder den här ytterligare informationen tillsammans med övriga data för att förbättra Microsofts produkter och tjänster för alla användare.

Vi använder inte den här informationen för att anpassa Microsofts produkter och tjänster (gäller båda nivåerna) om du inte ger oss tillåtelse via den separata inställningen Anpassad användningsupplevelse med diagnostikdata (beskrivs nedan).

Du kan ändra insamlingsnivån för diagnostikdata när du vill via Start > Inställningar > Sekretess > Feedback och diagnostik.

Anpassad användningsupplevelse med diagnostikdata

Om du väljer att aktivera anpassad användningsupplevelse använder vi några av de användningsdata som beskrivs i Diagnostik ovan för att anpassa din upplevelse av Windows och andra produkter och tjänster. Det här omfattar förslag på hur du anpassar och optimerar Windows. Du får även rekommendationer och erbjudanden om Windows-funktioner och appar, tjänster, maskinvara och kringutrustning som stöds. För att du ska få ut mesta möjliga av din enhet kan vi till exempel informera dig om funktioner du kanske inte känner till eller som är helt nya. Om du har problem med Windows-enheten kan du få förslag på lösningar. Du kan få möjlighet att anpassa låsskärmen med bilder. Du kan få tillgång till fler bilder av den typ som du tycker om eller färre av den som du inte tycker om. Om du strömmar en film i webbläsaren kan du få förslag på en app från Store som strömmar mer effektivt. Om diskutrymmet på hårddisken börjar ta slut kan Windows rekommendera att du provar onlinelagringstjänsten OneDrive eller köper till mer utrymme om så behövs. Omfattningen av dessa användningsdata beror på om du väljer fullständig eller grundläggande diagnostikrapportering. Om du till exempel väljer fullständig kommer dina data att omfatta användning av webbläsare och appar. Även om du aktiverar Anpassad användningsupplevelse använder vi dock inte dina indata som relaterar till kraschhändelser, tal, inmatning och pennanteckningar för att anpassa din upplevelse. Du kan inaktivera detta när du vill via Start > Inställningar > Sekretess > Feedback och diagnostik.

Relevanta annonser

Windows genererar ett unikt annonserings-ID för varje användare på en enhet som programutvecklare och annonsnätverk kan använda för att tillhandahålla mer relevanta annonser i appar. När annonserings-ID är aktiverat kan appar komma åt och använda det på ungefär samma sätt som webbplatser kan komma åt och använda en unik identifierare som lagrats i en cookie. Därför kan ditt annonserings-ID användas av apputvecklare (och på annonsnätverk som de arbetar med) för att tillhandahålla mer relevanta annonser och andra anpassade upplevelser i apparna. Du kan inaktivera detta när du vill via Start > Inställningar > Sekretess. Obs! Om du inaktiverar annonserings-ID minskas inte antalet annonser som visas, men det kan innebära att annonserna blir mindre intressanta och relevanta för dig. När du aktiverar funktionen igen återställs annonserings-ID:t.

Windows Defender SmartScreen

Windows Defender SmartScreen skickar data till Microsoft om webbplatser du besöker och filer du laddar ned för att kunna varna och skydda dig och din enhet från osäkert webbinnehåll och skadlig programvara. Eftersom vi strävar efter att skydda dig när du använder våra tjänster och tjänster från tredje part, aktiverar vi Windows Defender SmartScreen som standard. Du kan inaktivera Windows Defender SmartScreen när du vill via Start > Inställningar > Sekretess.
