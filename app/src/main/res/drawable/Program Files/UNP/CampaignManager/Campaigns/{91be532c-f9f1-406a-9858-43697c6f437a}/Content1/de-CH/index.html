Einführung

Beim Einrichten von Windows werden Sie zur Auswahl von Datenschutzeinstellungen aufgefordert. Auf der Einrichtungsseite schlagen wir Einstellungen für zuverlässigere und sicherere Verbindungen Ihres Gerätes vor. Diese Einstellungen sorgen für ein personalisiertes Erlebnis und unterstützen uns bei der Verbesserung unserer Dienste. Sie können Ihre Einstellungen jetzt auswählen oder diese jederzeit über Start > Einstellungen aktualisieren.

Die folgenden Informationen definieren die von uns entsprechend Ihrer Einstellungen gesammelten Daten. Lesen Sie die vollständige Microsoft-Datenschutzerklärung. Diese enthält Informationen darüber, welche Daten von Microsoft gespeichert werden und wie diese Daten bei der Nutzung von Windows verwendet werden (geben Sie aka.ms/privacy in ein Browser-Fenster ein). Wie in der Datenschutzerklärung von Microsoft angegeben, werden die erhobenen Daten in die USA und andere Länder gesendet und dort gespeichert.

Position

Die Positionsdienste von Microsoft stellen je nach Gerätefunktionen über GPS, nahe gelegene WLAN-Zugriffspunkte und/oder Mobilfunkmasten bzw. mit der IP-Adresse Standortinformationen für Windows-Geräte bereit.

Das Einschalten des Positionsdienstes ermöglicht bestimmten Apps, Diensten und Windows-Features, die Berechtigung für den Zugriff auf Positionsdaten mit der maximal auf Ihrem Gerät möglichen Genauigkeit anzufordern. Wenn Ihre Position von einer App oder einem Dienst mit Standortbestimmung verwendet wird, werden die Standortinformationen und Daten über den Positionsverlauf auf Ihrem Gerät gespeichert und zur Verbesserung der Positionsdienste in einem anonymisierten Format an Microsoft gesendet.

Wenn Sie sich mit Ihrem Microsoft-Konto angemeldet haben, wird zusätzlich Ihre letzte bekannte Standortinformation in der Cloud gespeichert. Diese Standortinformation steht anderen Apps oder Diensten, die Ihr Microsoft-Konto verwenden, zur Verfügung. Wenn das Gerät von alleine keinen guten Standort finden kann (wie z. B. in einem Haus oder Keller), kann es Ihren letzten bekannten Standort finden, der in der Cloud gespeichert ist.

Sie können unter Start > Einstellungen > Datenschutz > Position jederzeit den Standortzugriff deaktivieren und die Daten zum Standortverlauf Ihres Geräts löschen.

Wenn Sie über ein tragbares Gerät, z. B. einen Laptop verfügen, aktiviert die Positionserkennung auch die Funktion „Mein Gerät suchen”. Diese verwendet Ihre Positionsdaten, um Ihr Gerät wiederzufinden, wenn es verloren geht. Damit das Feature funktioniert, müssen Sie sich mit Ihrem Microsoft-Konto bei Windows anmelden. Sie können die Funktion jederzeit unter Start > Einstellungen > Update und Sicherheit > Mein Gerät suchen deaktivieren.

Spracherkennung

Windows bietet eine gerätebasierte Spracherkennungsfunktion (über die Desktop-App Windows-Spracherkennung) und einen cloudbasierten Spracherkennungsdienst, der zusammen mit Cortana in den Märkten und Regionen eingeführt wurde, in denen Cortana verfügbar ist. Um zu erfahren, welche Sprachen und Regionen die Spracherkennung derzeit unterstützt, suchen Sie in einem Browser oder in der Suchleiste nach „Cortana Regionen und Sprachen”. Beim Aktivieren der Spracherkennung sammelt Microsoft Ihre Sprachaufzeichnungen, um Sie mit Cloud-basierten Spracherkennungsdiensten in Cortana, Store-Apps und im Laufe der Zeit ebenfalls in anderen Bereichen von Windows zu unterstützen. Als Teil des Dienstes erfassen wir auch Informationen aus dem auf Ihrem Gerät erstellten Benutzerwörterbuch. Dieses Benutzerwörterbuch speichert von Ihnen eingegebene eindeutige Wörter wie Namen, um die Freihandzeichnung oder das Tippen zu verbessern. Sowohl die Stimmdaten als auch das Benutzerwörterbuch werden zusammen verwendet, um unsere Fähigkeit zu verbessern, die Sprechweise aller Benutzer zu erkennen. Sie können diese Funktion jederzeit unter Start > Einstellungen > Datenschutz > Spracherkennung, Freihand und Eingabe deaktivieren. Wenn Sie Cortana später den Zugriff auf Informationen, wie z. B. Kalender- und Kontaktdaten ermöglichen, kann Ihre Spracherkennungsfunktion personalisiert werden, um Ihnen einen besseren Service zu bieten. Beachten Sie, dass Sie jederzeit die gerätebasierte Spracherkennung durch die Eingabe von „Windows-Spracherkennung” in der Suchleiste und beim Ausführen der Desktop-App „Windows-Spracherkennung” verwenden können.

Diagnose

Beim Verwenden von Windows sammeln wir Diagnosedaten, die uns beim Identifizieren und Beheben von Problemen helfen und sicher stellen, dass Ihr Gerät auf dem neuesten Stand und sicher bleibt. Je nach den von Ihnen ausgewählten Einstellungen hilft die Diagnose auch dabei, unsere Produkte und Dienste zu verbessern und Ihnen personalisierte Erlebnisse anzubieten. Diese Daten werden mit einer oder mehreren eindeutigen Kennungen versehen, an Microsoft gesendet und gespeichert. Sie helfen uns dabei, einen einzelnen Benutzer auf einem einzelnen Gerät zu erkennen, Servicefälle des Gerätes zu verstehen und Nutzungsmuster zu speichern. Es gibt zwei Stufen von Diagnosedaten: Vollständig und Allgemein.

Allgemein sind Daten, die für den Betrieb von Windows unerlässlich sind. Mithilfe dieser Daten kann der sichere, stets aktuelle und einwandfreie Betrieb von Windows und Apps sichergestellt werden, da Microsoft technische Daten zum Gerät sowie Informationen zu den installierten Programmen und der ordnungsgemäßen Ausführung erhält. Diese Option umfasst die allgemeine Fehlerberichterstattung an Microsoft.

Vollständig umfasst alle allgemeinen Daten und Daten über die Nutzung von Apps, die Browsernutzung, die Verwendung bestimmter Funktionen oder Apps, den Zeitraum, für den Sie Apps und Features verwenden, wie oft Sie Windows-Hilfe und Support verwenden und welche Dienste Sie verwenden, um sich bei Apps anzumelden. Mit der Option „Vollständig” werden zusätzliche Diagnosedaten gesammelt. Diese umfassen beispielsweise den Speicherstatus Ihres Geräts im Fall eines System- oder App-Absturzes (diese Daten enthalten möglicherweise unbeabsichtigt Teile von Dokumenten, an denen Sie bei Auftreten eines Problems gearbeitet haben). Alle getippten oder per Hand eingegebenen Daten wie Worte oder Zeichen werden von den Identifikationsinformationen entfernt. Wir verwenden diese zusätzlichen Daten, um Microsoft-Produkte und Dienste für alle Benutzer zu verbessern.

Wir verwenden diese Daten beider Ebenen nicht, um Microsoft-Produkte oder Dienstleistungen zu personalisieren, es sei denn, Sie fordern uns dazu in den separaten Einstellungen der maßgeschneiderten Funktionen mit Diagnosedaten dazu auf (siehe unten).

Sie können die Erfassungsstufe jederzeit unter Start > Einstellungen > Datenschutz > Feedback und Diagnose ändern.

Maßgeschneiderte Funktionen mit Diagnosedaten

Wenn Sie maßgeschneiderte Funktionen aktivieren, verwenden wir einige der oben unter „Diagnose” beschriebenen Nutzungsdaten, um Ihre Erfahrung mit Windows und anderen Produkten und Diensten zu personalisieren. Dazu gehören Vorschläge zum Anpassen und Optimieren von Windows sowie Empfehlungen und Angebote über Windows-Funktionen und unterstützte Apps, Dienste, Hardware und Peripheriegeräte. Um Ihr Gerät optimal zu unterstützen, informieren wir Sie eventuell über Funktionen, die Sie möglicherweise nicht kennen oder die neu sind. Wenn Sie ein Problem mit Ihrem Windows-Gerät haben, wird Ihnen dort möglicherweise eine Lösung angeboten. Sie haben eventuell die Chance, Ihren Sperrbildschirm mit Bildern anzupassen oder ähnliche gewünschte Bilder auszusuchen, oder weniger erwünschte Bilder anzuzeigen. Wenn Sie einen Film in Ihrem Browser streamen, kann eine App aus dem Store empfohlen werden, die effizienter streamt. Wenn Sie alternativ nicht genügend Speicherplatz auf der Festplatte haben, kann Windows empfehlen, den OneDrive-Onlinespeicher-Dienst auszuprobieren oder den notwendigen zusätzlichen Speicherplatz zu erwerben. Der Umfang der Nutzungsdaten hängt davon ab, ob Sie vollständige oder allgemeine Diagnosedaten ausgewählt haben. Wenn Sie z. B. „Vollständig” auswählen, enthalten Ihre Nutzungsdaten Browser- und App-Nutzungsinformationen. Wenn Sie maßgeschneiderte Funktionen aktivieren, werden Ihre Absturzdaten, Spracherkennungsdaten und Freihand- oder Eingabedaten trotzdem nicht verwendet. Sie können diese Funktion jederzeit unter Start > Einstellungen > Datenschutz > Feedback und Diagnose deaktivieren.

Relevante Werbung

Windows erzeugt für jeden Benutzer auf einem Gerät eine eindeutige Werbe-ID, die von Anwendungsentwicklern und Werbenetzwerken zur Anzeige von relevanter Werbung in Apps genutzt werden kann. Wenn die Werbe-ID aktiviert ist, können Apps auf diese zugreifen und sie auf ähnliche Weise wie Websites verwenden, die auf eine eindeutige, in einem Cookie gespeicherte Kennung, zugreifen. Daher kann Ihre Werbe-ID von App-Entwicklern (sowie den mit ihnen verbundenen Werbenetzwerken) verwendet werden, um eine relevantere Werbung und andere personalisierte Erfahrungen mit deren Apps anzubieten. Sie können diese Funktion jederzeit unter Start > Einstellungen > Datenschutz deaktivieren. Beachten Sie, dass das Deaktivieren der Werbe-ID nicht die Anzahl der Werbeanzeigen reduziert. Die Anzeigen können jedoch weniger relevant sein. Wenn Sie diese Option wieder aktivieren, wird die Werbe-ID zurückgesetzt.

Windows Defender SmartScreen

Windows Defender SmartScreen sendet an Microsoft Daten über die von Ihnen besuchten Websites sowie Dateien, die Sie herunterladen, um Sie und Ihr Gerät vor unsicheren Webinhalten oder Schadsoftware zu warnen und zu schützen. Da wir uns bemühen, Sie beim Verwenden unserer Dienste und der von Drittanbietern zu schützen, ist Windows Defender SmartScreen standardmäßig aktiviert. Sie können Windows Defender SmartScreen jederzeit unter Start > Einstellungen > Datenschutz deaktivieren.
