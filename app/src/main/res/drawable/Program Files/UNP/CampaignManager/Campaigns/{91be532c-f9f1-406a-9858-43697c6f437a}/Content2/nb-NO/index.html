Kommer snart på Windows 10

Noe av det beste med Windows 10, er at operativsystemet forbedres hele tiden. Nye funksjoner. Oppdatert sikkerhet. Det er enkelt å holde deg oppdatert. Når automatiske oppdateringer er aktivert, får du nye Windows 10-funksjoner og -sikkerhetsforbedringer så snart de er tilgjengelige. Tjenesten er kostnadsfri så lenge enheten støttes.*

De nyeste innovative funksjonene kommer i Windows 10 Creators Update. Denne oppdateringen gir deg de nyeste sikkerhetsforbedringene, pluss en rekke nye funksjoner, som for eksempel:

  • 3D in Windows 10 med et nytt Paint 3D-program som kan brukes for å skape og dele ting med en ekstra dimensjon
  • PC-spilling med 4K spill, spillmodus og innebygd spillkringkasting
  • Sikkerhetsinstrumentbord og tilstandsvarsel for enheten

Windows 10-opplevelsen blir stadig bedre!

Enkelte funksjoner er avhengige av maskinvaren. * Kostnader for Internett-tilgang kan påløpe.

Innføring

Når du konfigurerer Windows, ber vi deg velge innstillinger for personvern. På konfigurasjonsskjermen anbefaler vi innstillingene som gjør enheten din mest mulig trygg og pålitelig når du kobler til tjenester, slik at du får en mer personlig tilpasset brukeropplevelse, og slik at vi kan forbedre tjenestene våre. Du kan velge innstillingene nå, eller oppdatere dem når som helst ved å gå til Start > Innstillinger.

Informasjonen nedenfor forklarer hvilke typer data som samles inn, og hvordan de brukes, avhengig av innstillingene du velger. Sørg for å lese gjennom hele personvernerklæringen til Microsoft hvis du vil ha mer informasjon om alle persondataene vi samler inn, og hvordan de brukes når du bruker Windows (skriv aka.ms/privacy i et hvilket som helst nettleservindu for å gjøre dette). Dataene vi samler inn, sendes til og lagres i USA eller andre land i henhold til personvernerklæringen for Microsoft.

Posisjon

Microsofts posisjonstjeneste gir posisjonsinformasjon til Windows-enheter ved hjelp av en kombinasjon av globale posisjoneringstjenester (GPS), trådløse tilgangspunkt i nærheten, telefontårn og IP-adressen din, avhengig av enhetens funksjoner.

Hvis du slår på Posisjon, kan enkelte apper, tjenester og Windows-funksjoner be om tillatelse til å få tilgang til og bruke posisjonsdataene dine for å levere stedsavhengige tjenester på det nøyaktighetsnivået som enheten støtter. Når posisjonen din brukes av en stedsfølsom app eller tjeneste, lagres posisjonsinformasjon og loggen for nylige posisjoner på enheten, og disse dataene sendes til Microsoft i et anonymisert format for å forbedre posisjonstjenestene.

Hvis du er logget på med Microsoft-kontoen din, lagres den siste kjente posisjonsinformasjonen din i skyen, og disse dataene er tilgjengelige for andre apper eller tjenester som bruker Microsoft-kontoen din på de ulike enhetene dine. Hvis enheten ikke klarer å finne en god posisjon selv (f.eks. i en bygning eller kjeller), kan den bruke den siste kjente fungerende posisjonen som er lagret i skyen.

Du kan når som helst slå av posisjonstilgangen og tømme posisjonsloggen for enheten ved å gå til Start > Innstillinger > Personvern > Posisjon.

Hvis du har en bærbar enhet, f.eks. en bærbar datamaskin, aktiveres Finn enheten-funksjonen automatisk når du slår på posisjonstjenester. Denne funksjonen bruker posisjonsdataene dine til å hjelpe deg å finne enheten hvis du mister den. For at denne funksjonen skal fungere, må du logge på Windows med Microsoft-kontoen din. Du kan deaktivere denne funksjonen når som helst ved å gå til Start > Innstillinger > Oppdatering og sikkerhet > Finn enheten.

Talegjenkjenning

Windows leverer både en enhetsbasert funksjon for talegjenkjenning (som er tilgjengelig via skrivebord-appen Windows Talegjenkjenning) og en skybasert talegjenkjenningstjeneste som ble lansert sammen med Cortana i markeder og regioner der Cortana er tilgjengelig. Hvis du vil vite hvilke språk og områder som støttes av talegjenkjenningsfunksjonen, kan du søke etter "Cortanas områder og språk" i en hvilken som helst nettleser, eller i søkefeltet. Hvis du slår på Talegjenkjenning, kan Microsoft samle inn og bruke stemmeinnspillingene dine for å gi deg skybaserte talegjenkjenningstjenester i Cortana, støttede Store-apper, og etter hvert også i andre deler av Windows. Som en del av denne tjenesten, samler vi også inn informasjon fra brukerordlisten som er opprettet på enheten. Denne brukerordlisten lagrer unike ord som f.eks. navn du skriver, noe som bidrar til å gjøre tasting og håndskrift mer nøyaktig. Både stemmedataene og brukerordlisten brukes i samlelisten for å forbedre evnen til å gjenkjenne tale for alle brukere. Du kan deaktivere denne funksjonen når som helst ved å gå til Start > Innstillinger > Personvern > Tale, håndskrift og inntasting. Hvis du senere vil gi Cortana tilgang til informasjon som kalender- og kontaktdata, kan talegjenkjenningsopplevelsen tilpasses slik at den fungerer bedre for deg. Vær oppmerksom på at du alltid kan bruke enhetsbaserte talegjenkjenning ved å skrive inn "Windows Talegjenkjenning" i søkefeltet og kjøre skrivebordsappen Windows Talegjenkjenning.

Diagnostikk

Mens du bruker Windows, vil vi samle inn diagnostisering data som hjelper oss med å identifisere og løse problemer, og at enhetene dine er oppdaterte og sikre. Avhengig av innstillingene du velger, kan diagnosefunksjonen også hjelpe oss å forbedre produktene og tjenestene våre, samt gi deg tilpassede brukeropplevelser. Informasjonen overføres til Microsoft og lagres med én eller flere unike identifikatorer som hjelper oss å gjenkjenne en individuell bruker på en individuell enhet, og å forstå serviceproblemer og bruksmønstre på enheten. Det finnes to nivåer av diagnosedata: Fullstendig og Grunnleggende.

Grunnleggende data er data som er viktige for at Windows skal fungere. Disse dataene bidrar til å holde Windows og apper sikre og oppdaterte, og sørger for at de kjører som de skal, ved at Microsoft får kjennskap til kapasiteten til enheten din, hva som er installert på den, og om Windows fungerer som det skal. Dette alternativet omfatter grunnleggende feilrapportering til Microsoft.

Fullstendige data omfatter alt som samles inn som Grunnleggende data, pluss data om bruk av appen, bruk av nettleseren, hvordan og hvor lenge du bruker bestemte funksjoner eller apper, hvilke apper og funksjoner du bruker oftest, hvor ofte du bruker Windows Hjelp og brukerstøtten, samt hvilke tjenester du bruker for å logge deg på apper. På Fullstendig-nivå samler Microsoft inn ytterligere diagnosedata, inkludert minnetilstanden til enheten når det oppstår system- eller appkrasj (dette kan utilsiktet inkludere deler av en fil du brukte da problemet oppsto). Identifiserende informasjon fjernes fra inntastings- og håndskriftdataene dine (f.eks. ord eller tegn). Vi bruker disse tilleggsdataene i samlefunksjonen for å forbedre Microsofts produkter og tjenester for alle brukere.

Vi bruker ikke disse dataene for å tilpasse Microsoft-produkter eller -tjenester med mindre du lar oss gjøre det via de separate innstillingene for diagnosedata for tilpassede brukeropplevelser (se nedenfor). Dette gjelder på begge nivåene.

Du kan justere nivået for innsamling av diagnosedata når som helst ved å gå til Start > Innstillinger > Personvern > Tilbakemelding og diagnose.

Skreddersydde brukeropplevelser med diagnosedata

Hvis du velger å aktivere skreddersydde brukeropplevelser, bruker vi noen av bruksdataene som er beskrevet ovenfor under "Diagnostikk" for å tilpasse brukeropplevelsene dine i Windows og andre produkter og tjenester. Dette omfatter forslag til hvordan du kan tilpasse og optimalisere Windows, samt anbefalinger av og tilbud om Windows-funksjoner og støttede programmer, tjenester, maskinvare og enheter. For å hjelpe deg å få mest mulig ut av enheten kan det f.eks. hende vi forteller deg om funksjoner du kanskje ikke kjenner til, eller som er nye. Hvis du har et problem med Windows-enheten din, kan du bli tilbudt en løsning. Du kan få mulighet til å tilpasse låseskjermen med bilder, eller få se flere bilder av typen du liker, eller færre av de du ikke liker. Hvis du strømmer en film i nettleseren, kan du bli anbefalt en Store-app som strømmer mer effektivt. Hvis du ikke har nok plass på harddisken, kan Windows komme til å anbefale at du prøver OneDrive-lagringsplass på nettet, eller at du kan kjøpe mer lagringsplass. Omfanget av bruksdataene avhenger av om du velger Fullstendig eller Grunnleggende diagnoserapportering. Hvis du velger Fullstendig, inkluderes f.eks. bruksdata for nettleseren og appene du bruker. Selv om du aktiverer skreddersydde brukeropplevelser, bruker vi ikke krasj-, tale-, inndata- eller håndskriftsdataene dine for å tilpasse brukeropplevelsen din. Du kan deaktivere denne funksjonen når som helst ved å gå til Start > Innstillinger > Personvern > Tilbakemelding og diagnose.

Relevante annonser

Windows genererer en unik annonse-ID for hver av enhetens brukere, og denne kan utviklere av apper og annonsenettverk bruke til å tilby mer relevant annonsering i apper. Når annonse-ID-en er aktivert, kan apper få tilgang til og bruke annonse-ID-en på omtrent samme måte som nettsteder kan få tilgang til og bruke en unik identifikator som er lagret i en informasjonskapsel. Dermed kan annonse-ID-en brukes av apputviklere (og annonsenettverkene de samarbeider med) for å tilby mer relevante annonser og tilpassede opplevelser gjennom appene sine. Du kan deaktivere denne funksjonen når som helst ved å gå til Start > Innstillinger > Personvern. Du ser ikke færre annonser ved å deaktivere annonse-ID, men det kan innebære at annonsene er mindre interessante og relevante for deg. Hvis du slår på funksjonen igjen, nullstilles annonse-ID.

Windows Defender SmartScreen

Windows Defender SmartScreen sender data til Microsoft om nettstedene du besøker og filene du laster ned for å kunne hjelpe med å advare og beskytte deg og denne enheten mot usikkert nettleserinnhold eller ondsinnet programvare. Siden vi gjør vårt beste for å beskytte deg når du bruker tjenestene våre og tjenester fra tredjeparter, aktiverer vi Windows Defender SmartScreen som standard. Du kan slå av Windows Defender SmartScreen når som helst ved å gå til Start > Innstillinger > Personvern.
