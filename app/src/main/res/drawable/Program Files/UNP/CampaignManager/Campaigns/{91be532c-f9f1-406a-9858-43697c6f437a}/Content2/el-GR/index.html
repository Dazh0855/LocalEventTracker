Κωδικός σφάλματος:

Προσεχώς στα Windows 10

Ένα από τα καλύτερα χαρακτηριστικά των Windows 10 είναι ότι βελτιώνονται συνεχώς. Νέες δυνατότητες. Ενημερωμένη ασφάλεια. Πανεύκολη εγκατάσταση ενημερώσεων. Με τις αυτόματες ενημερώσεις ενεργοποιημένες, λαμβάνετε τις νέες δυνατότητες και τις βελτιώσεις ασφαλείας των Windows 10 μόλις κυκλοφορούν, χωρίς κόστος για την υποστηριζόμενη διάρκεια ζωής της συσκευής σας.*

Η επόμενη φάση καινοτομιών έρχεται στα Windows 10 Αναβάθμιση Δημιουργικότητας. Αυτή η ενημέρωση παρέχει τις τελευταίες βελτιώσεις ασφαλείας και μια ποικιλία εκπληκτικών νέων δυνατοτήτων, όπως:

  • 3D στα Windows 10 που περιλαμβάνει τη νέα εφαρμογή Ζωγραφική 3D, με την οποία η δημιουργία και η κοινή χρήση περνούν σε μια εντελώς νέα διάσταση
  • Παιχνίδι σε υπολογιστή με παιχνίδια 4K, λειτουργία παιχνιδιού και ενσωματωμένη μετάδοση παιχνιδιών
  • Οθόνη πίνακα εργαλείων ασφαλείας και σύμβουλος εύρυθμης λειτουργίας συσκευής

Η εμπειρία Windows 10 γίνεται ολοένα και καλύτερη!

Ορισμένες δυνατότητες εξαρτώνται από το υλικό. * Ενδέχεται να ισχύουν χρεώσεις πρόσβασης στο Internet.

Εισαγωγή

Κατά την εγκατάσταση των Windows, σας ζητούμε να επιλέξετε τις ρυθμίσεις που σχετίζονται με τα προσωπικά σας δεδομένα. Στην οθόνη εγκατάστασης, προτείνουμε τις ρυθμίσεις που θεωρούμε κατάλληλες και που θα βοηθήσουν να αυξηθεί η αξιοπιστία και η ασφάλεια της συσκευής σας κατά τη σύνδεση σε υπηρεσίες, να σας παρέχουμε μια πιο εξατομικευμένη εμπειρία, καθώς και να βελτιώσουμε τις υπηρεσίες μας. Μπορείτε να επιλέξετε τις ρυθμίσεις σας τώρα ή να τις ενημερώσετε οποιαδήποτε άλλη στιγμή μεταβαίνοντας στην περιοχή Έναρξη > Ρυθμίσεις.

Οι παρακάτω πληροφορίες εξηγούν ποια δεδομένα συλλέγουμε και πώς αυτά χρησιμοποιούνται, ανάλογα με τις ρυθμίσεις που επιλέγετε. Διαβάστε την πλήρη Δήλωση προστασίας προσωπικών δεδομένων της Microsoft για περισσότερες πληροφορίες σχετικά με όλα τα προσωπικά δεδομένα που συλλέγουμε και τον τρόπο με τον οποίο αυτά χρησιμοποιούνται, όταν χρησιμοποιείτε τα Windows (πληκτρολογήστε aka.ms/privacy στο παράθυρο οποιουδήποτε προγράμματος περιήγησης). Τα δεδομένα που συλλέγουμε αποστέλλονται και αποθηκεύονται στις ΗΠΑ και σε άλλες χώρες/περιοχές, όπως καθορίζεται στη Δήλωση προστασίας προσωπικών δεδομένων της Microsoft.

Τοποθεσία

Η υπηρεσία τοποθεσίας της Microsoft παρέχει πληροφορίες τοποθεσίας σε συσκευές Windows, με τη συνδυαστική χρήση του παγκόσμιου συστήματος εντοπισμού θέσης (GPS), κοντινών σημείων ασύρματης πρόσβασης, πύργων κινητής τηλεφωνίας και της διεύθυνσης IP σας, ανάλογα με τις δυνατότητες της συσκευής σας.

Η ενεργοποίηση της υπηρεσίας τοποθεσίας επιτρέπει σε ορισμένες εφαρμογές, υπηρεσίες και δυνατότητες των Windows να ζητούν άδεια για την πρόσβαση και τη χρήση των δεδομένων τοποθεσίας σας, με σκοπό την παροχή υπηρεσιών με επίγνωση τοποθεσίας στο επίπεδο ακρίβειας που υποστηρίζει η συσκευή σας. Όταν η τοποθεσία σας χρησιμοποιείται από μια εφαρμογή ή υπηρεσία με επίγνωση τοποθεσίας, οι πληροφορίες τοποθεσίας και το ιστορικό πρόσφατων τοποθεσιών αποθηκεύονται στη συσκευή σας και αποστέλλονται στη Microsoft σε ανωνυμοποιημένη μορφή για τη βελτίωση των υπηρεσιών τοποθεσίας.

Επιπλέον, αν έχετε συνδεθεί με το λογαριασμό Microsoft, οι πληροφορίες τελευταίας γνωστής σωστής τοποθεσίας σας αποθηκεύονται στο cloud και παρέχονται σε άλλες εφαρμογές ή υπηρεσίες που χρησιμοποιούν το λογαριασμό Microsoft σε όλες τις συσκευές σας. Αν η συσκευή σας δεν μπορεί να εντοπίσει μια καλή τοποθεσία από μόνη της (για παράδειγμα, στο εσωτερικό ενός κτηρίου ή σε ένα υπόγειο), μπορεί να χρησιμοποιήσει την τελευταία γνωστή καλή τοποθεσία που είναι αποθηκευμένη στο cloud.

Μπορείτε να απενεργοποιήσετε την πρόσβαση στην υπηρεσία τοποθεσίας και να εκκαθαρίσετε το ιστορικό τοποθεσιών της συσκευής σας ανά πάσα στιγμή από την περιοχή Έναρξη > Ρυθμίσεις > Προστασία προσωπικών δεδομένων > Τοποθεσία.

Αν διαθέτετε φορητή συσκευή, όπως φορητό υπολογιστή, με την ενεργοποίηση της υπηρεσίας τοποθεσίας θα ενεργοποιήσετε και τη δυνατότητα "Εύρεση της συσκευής μου", η οποία χρησιμοποιεί τα δεδομένα τοποθεσίας σας, για να σας βοηθήσει να εντοπίσετε τη συσκευή σας, αν τη χάσετε. Για να λειτουργήσει αυτή η δυνατότητα, πρέπει να συνδεθείτε στα Windows με το λογαριασμό Microsoft. Μπορείτε να απενεργοποιήσετε αυτή τη δυνατότητα ανά πάσα στιγμή από την περιοχή Έναρξη > Ρυθμίσεις > Ενημέρωση και ασφάλεια > Εύρεση της συσκευής μου.

Αναγνώριση ομιλίας

Τα Windows παρέχουν μια δυνατότητα αναγνώρισης ομιλίας που βασίζεται στη συσκευή (διατίθεται μέσω της εφαρμογής Αναγνώρισης ομιλίας των Windows για υπολογιστή), καθώς και μια υπηρεσία αναγνώρισης ομιλίας που βασίζεται στο cloud, η οποία κυκλοφόρησε μαζί με την Cortana στις αγορές και τις περιοχές όπου η Cortana είναι διαθέσιμη. Για να μάθετε ποιες γλώσσες και ποιες περιοχές υποστηρίζονται προς το παρόν, αναζητήστε τη φράση "Περιοχές και γλώσσες της Cortana" σε οποιοδήποτε πρόγραμμα περιήγησης ή στη γραμμή αναζήτησης. Η ενεργοποίηση της ρύθμισης Αναγνώρισης ομιλίας επιτρέπει στη Microsoft να συλλέγει και να χρησιμοποιεί τις εγγραφές φωνής σας, για να σας παρέχει υπηρεσίες αναγνώρισης ομιλίας που βασίζονται στο cloud στην Cortana, σε υποστηριζόμενες εφαρμογές του Store και σε άλλες υπηρεσίες των Windows, με την πάροδο του χρόνου. Στο πλαίσιο αυτής της υπηρεσίας, συλλέγουμε επίσης πληροφορίες από το λεξικό χρήστη που έχετε δημιουργήσει στη συσκευή σας. Σε αυτό το λεξικό χρήστη έχουν αποθηκευτεί μοναδικές λέξεις, όπως κάποια ονόματα που έχετε γράψει, και σας βοηθά να πληκτρολογείτε και να γράφετε με μεγαλύτερη ακρίβεια. Τα φωνητικά δεδομένα και το λεξικό χρήστη χρησιμοποιούνται συγκεντρωτικά, με σκοπό να βελτιωθεί η ικανότητά μας να αναγνωρίζουμε σωστά την ομιλία κάθε χρήστη. Μπορείτε να απενεργοποιήσετε αυτή τη δυνατότητα ανά πάσα στιγμή από την περιοχή Έναρξη > Ρυθμίσεις > Προστασία προσωπικών δεδομένων > Φωνητικές εντολές, γραφή με μελάνη και πληκτρολόγηση. Αν αργότερα επιτρέψετε στην Cortana να αποκτά πρόσβαση σε πληροφορίες όπως το ημερολόγιο και οι επαφές σας, η εμπειρία αναγνώρισης ομιλίας μπορεί να εξατομικευτεί ώστε να λειτουργεί καλύτερα για εσάς. Λάβετε υπόψη ότι μπορείτε να χρησιμοποιήσετε την αναγνώριση ομιλίας που βασίζεται στη συσκευή ανά πάσα στιγμή πληκτρολογώντας "Αναγνώριση ομιλίας των Windows" στη γραμμή αναζήτησης και ανοίγοντας την εφαρμογή Αναγνώρισης ομιλίας των Windows για υπολογιστή.

Διαγνωστικός έλεγχος

Καθώς χρησιμοποιείτε τα Windows, συλλέγουμε δεδομένα διαγνωστικών που μας βοηθούν να αναγνωρίζουμε και να επιλύουμε προβλήματα, καθώς και να διατηρούμε τις συσκευές σας ενημερωμένες και ασφαλείς. Ανάλογα με τις ρυθμίσεις που θα επιλέξετε, ο διαγνωστικός έλεγχος μπορεί επίσης να μας βοηθήσει να βελτιώσουμε τα προϊόντα και τις υπηρεσίες μας και να σας προσφέρουμε εξατομικευμένες εμπειρίες. Αυτά τα δεδομένα μεταδίδονται στη Microsoft και αποθηκεύονται με ένα ή περισσότερα μοναδικά αναγνωριστικά που μπορούν να μας βοηθήσουν να αναγνωρίσουμε έναν μεμονωμένο χρήστη σε μία συσκευή και να κατανοήσουμε τα ζητήματα των υπηρεσιών της συσκευής και τα μοτίβα χρήσης. Υπάρχουν δύο επίπεδα διαγνωστικών δεδομένων: Πλήρη και Βασικά.

Τα Βασικά δεδομένα είναι ζωτικά για τη λειτουργία των Windows. Μας επιτρέπουν να γνωρίζουμε τις δυνατότητες της συσκευής σας, τα προγράμματα που είναι εγκατεστημένα και αν τα Windows λειτουργούν σωστά, ώστε να παραμένουν ασφαλή, ενημερωμένα και λειτουργικά τόσο τα Windows όσο και οι εφαρμογές. Αυτή η επιλογή περιλαμβάνει την αναφορά βασικών σφαλμάτων στη Microsoft.

Τα Πλήρη δεδομένα περιλαμβάνουν όλα τα στοιχεία που συλλέγονται με τα Βασικά δεδομένα, καθώς και δεδομένα σχετικά με τη χρήση των εφαρμογών, τη χρήση των προγραμμάτων περιήγησης, τον τρόπο και τη διάρκεια χρήσης ορισμένων δυνατοτήτων ή εφαρμογών, τις εφαρμογές και τις δυνατότητες που χρησιμοποιείτε συχνότερα, τη συχνότητα χρήσης της Βοήθειας και της υποστήριξης των Windows και τις υπηρεσίες που χρησιμοποιείτε για να συνδέεστε σε εφαρμογές. Με τα Πλήρη δεδομένα, η Microsoft συλλέγει πρόσθετα διαγνωστικά δεδομένα, συμπεριλαμβανομένης της κατάστασης μνήμης της συσκευής τη στιγμή που παρουσιάστηκε ένα σφάλμα συστήματος ή εφαρμογής (στα οποία ενδέχεται να περιλαμβάνονται τμήματα ενός αρχείου που χρησιμοποιούσατε όταν παρουσιάστηκε το πρόβλημα χωρίς κάποια σκοπιμότητα). Τυχόν αναγνωριστικές πληροφορίες θα καταργούνται από τα πληκτρολογημένα και χειρόγραφα δεδομένα εισόδου, όπως λέξεις ή χαρακτήρες. Θα χρησιμοποιούμε αυτά τα πρόσθετα δεδομένα συγκεντρωτικά, με σκοπό τη βελτίωση των προϊόντων και των υπηρεσιών της Microsoft για όλους τους χρήστες.

Στην περίπτωση και των δύο επιπέδων, τα δεδομένα αυτά δεν χρησιμοποιούνται για σκοπούς εξατομίκευσης των προϊόντων ή των υπηρεσιών της Microsoft, εκτός εάν μας επιτρέψετε κάτι τέτοιο μέσω της ξεχωριστής ρύθμισης εξατομικευμένων εμπειριών με τη βοήθεια διαγνωστικών δεδομένων (όπως περιγράφεται παρακάτω).

Μπορείτε να προσαρμόσετε το επίπεδο συλλογής διαγνωστικών δεδομένων ανά πάσα στιγμή από την περιοχή Έναρξη > Ρυθμίσεις > Προστασία προσωπικών δεδομένων > Σχόλια και διαγνωστικός έλεγχος.

Εξατομικευμένες εμπειρίες με τη βοήθεια διαγνωστικών δεδομένων

Αν επιλέξετε να ενεργοποιήσετε τις εξατομικευμένες εμπειρίες, θα χρησιμοποιήσουμε ορισμένα από τα δεδομένα χρήσης που περιγράφονται παραπάνω στην ενότητα "Διαγνωστικός έλεγχος", για να εξατομικεύσουμε την εμπειρία σας με τα Windows και άλλα προϊόντα και υπηρεσίες. Αυτό περιλαμβάνει προτάσεις σχετικά με τον τρόπο προσαρμογής και βελτιστοποίησης των Windows, καθώς και προτάσεις και προσφορές για δυνατότητες των Windows και υποστηριζόμενες εφαρμογές, υπηρεσίες, υλικό και περιφερειακές συσκευές. Για παράδειγμα, για να αξιοποιήσετε στο έπακρο τη συσκευή σας, μπορεί να σας παράσχουμε πληροφορίες για νέες δυνατότητες ή δυνατότητες που μπορεί να μην γνωρίζετε ακόμα. Αν αντιμετωπίσετε πρόβλημα με τη συσκευή Windows, μπορεί να σας προσφέρουμε μια λύση. Μπορεί να σας δώσουμε την ευκαιρία να προσαρμόσετε την οθόνη κλειδώματος με εικόνες ή μπορεί να σας δείχνουμε περισσότερες εικόνες από εκείνες που σας αρέσουν ή λιγότερες από εκείνες που δεν σας αρέσουν. Αν αναπαράγετε μια ταινία μέσω ροής στο πρόγραμμα περιήγησής σας, μπορεί να σας προτείνουμε μια πιο αποτελεσματική εφαρμογή ροής από το Store. Εναλλακτικά, αν ο χώρος στον σκληρό σας δίσκο εξαντλείται, τα Windows μπορεί να σας προτείνουν να δοκιμάσετε την υπηρεσία online χώρου αποθήκευσης OneDrive ή να αγοράσετε περισσότερο χώρο αποθήκευσης, αν χρειάζεστε. Το εύρος των δεδομένων χρήσης θα εξαρτηθεί από το αν θα επιλέξετε να αναφέρονται Πλήρη ή Βασικά διαγνωστικά δεδομένα. Για παράδειγμα, αν επιλέξετε τα Πλήρη δεδομένα, στα δεδομένα θα περιλαμβάνεται και η χρήση των προγραμμάτων περιήγησης και των εφαρμογών. Ωστόσο, ακόμα και αν ενεργοποιήσετε τις εξατομικευμένες εμπειρίες, δεν θα χρησιμοποιήσουμε τα δεδομένα σας που σχετίζονται με τα σφάλματα, την ομιλία, την πληκτρολόγηση ή τη γραφή για σκοπούς εξατομίκευσης. Μπορείτε να απενεργοποιήσετε αυτή τη δυνατότητα ανά πάσα στιγμή από την περιοχή Έναρξη > Ρυθμίσεις > Προστασία προσωπικών δεδομένων > Σχόλια και διαγνωστικός έλεγχος.

Σχετικές διαφημίσεις

Τα Windows δημιουργούν ένα μοναδικό αναγνωριστικό διαφήμισης για κάθε χρήστη σε μια συσκευή, το οποίο οι προγραμματιστές εφαρμογών και τα διαφημιστικά δίκτυα μπορούν να χρησιμοποιούν για να παρέχουν πιο συναφείς διαφημίσεις στις εφαρμογές. Όταν είναι ενεργοποιημένο το αναγνωριστικό διαφήμισης, οι εφαρμογές μπορούν να έχουν πρόσβαση σε αυτό και να το χρησιμοποιούν με τον ίδιο τρόπο με τον οποίο οι τοποθεσίες web μπορούν να έχουν πρόσβαση σε ένα μοναδικό αναγνωριστικό που είναι αποθηκευμένο σε ένα cookie και να το χρησιμοποιούν. Ως εκ τούτου, το αναγνωριστικό διαφήμισης μπορεί να χρησιμοποιηθεί από προγραμματιστές εφαρμογών (και τα διαφημιστικά δίκτυα με τα οποία συνεργάζονται) για την παροχή πιο συναφών διαφημίσεων και άλλων εξατομικευμένων εμπειριών στις εφαρμογές τους. Μπορείτε να απενεργοποιήσετε αυτήν τη δυνατότητα ανά πάσα στιγμή από την περιοχή Έναρξη > Ρυθμίσεις > Προστασία προσωπικών δεδομένων. Λάβετε υπόψη ότι η απενεργοποίηση του αναγνωριστικού διαφήμισης δεν θα μειώσει τον αριθμό των διαφημίσεων που θα βλέπετε, αλλά υπάρχει περίπτωση οι διαφημίσεις αυτές να είναι λιγότερο ενδιαφέρουσες για εσάς. Αν την ενεργοποιήσετε ξανά, θα γίνει επαναφορά του αναγνωριστικού διαφήμισης.

Windows Defender SmartScreen

Το Windows Defender SmartScreen αποστέλλει στη Microsoft δεδομένα για τις τοποθεσίες web που επισκέπτεστε και τα αρχεία που κατεβάζετε, ώστε να σας προειδοποιεί και να προστατεύει τόσο εσάς όσο και τη συσκευή σας από μη ασφαλές περιεχόμενο web ή κακόβουλο λογισμικό. Καθώς καταβάλλουμε κάθε δυνατή προσπάθεια να σας προστατεύουμε όταν χρησιμοποιείτε τις υπηρεσίες μας και υπηρεσίες τρίτων, το Windows Defender SmartScreen είναι ενεργοποιημένο από προεπιλογή. Μπορείτε να απενεργοποιήσετε το Windows Defender SmartScreen ανά πάσα στιγμή από την περιοχή Έναρξη > Ρυθμίσεις > Προστασία προσωπικών δεδομένων.

Τα Windows θα ενημερωθούν σύντομα

Απλώς μια υπενθύμιση: η έκδοση Windows 10 Αναβάθμιση Δημιουργικότητας είναι έτοιμη και σε αναμονή. 

Προετοιμάζουμε την ενημέρωση για τη συσκευή σας. Η λήψη θα εκτελεστεί στο παρασκήνιο, ώστε να συνεχίσετε κανονικά την εργασία σας. Η ενημέρωση θα εγκατασταθεί με βάση τις ρυθμίσεις του Windows Update.

Αν αυτή η στιγμή δεν είναι κατάλληλη, επιλέξτε Όχι τώρα και θα επικοινωνήσουμε ξανά μαζί σας αργότερα.

Όχι τώρα

Επιλογή ρυθμίσεων προσωπικών δεδομένων για τη συσκευή

Η Microsoft σάς παρέχει τον έλεγχο της προστασίας των προσωπικών δεδομένων σας. Οι ακόλουθες προτάσεις βασίζονται στις τρέχουσες ρυθμίσεις σας. Οι αλλαγές που θα κάνετε θα ισχύσουν για όλους τους λογαριασμούς σε αυτήν τη συσκευή, όταν εγκαταστήσετε την έκδοση Windows 10 Αναβάθμιση Δημιουργικότητας. Πραγματοποιήστε κύλιση για να προβάλετε όλες τις ρυθμίσεις.

ΕνεργόΑνενεργό
Τοποθεσία

Λάβετε εμπειρίες βασισμένες στην τοποθεσία, όπως τοπική πρόγνωση καιρού και οδηγίες για τα αγαπημένα σας μέρη. Επιτρέψτε στα Windows και στις εφαρμογές να ζητούν την τοποθεσία σας και στείλτε στη Microsoft δεδομένα τοποθεσίας για τη βελτίωση των υπηρεσιών τοποθεσίας.

Τα Windows και οι εφαρμογές δεν μπορούν να χρησιμοποιήσουν την τοποθεσία σας για να παρέχουν πληροφορίες, όπως τοπική πρόγνωση του καιρού, οδηγίες και τη δυνατότητα Εύρεση της συσκευής μου.

ΕνεργόΑνενεργό
Αναγνώριση ομιλίας

Μιλήστε στην Cortana και σε εφαρμογές του Store που υποστηρίζουν αναγνώριση φωνής. Στείλτε στη Microsoft τα φωνητικά δεδομένα εισόδου σας για σκοπούς βελτίωσης των υπηρεσιών φωνητικών εντολών.

Δεν μπορείτε να μιλήσετε στην Cortana ή σε ορισμένες εφαρμογές του Store.

ΠλήρηΒασικά
Διαγνωστικός έλεγχος

Βοηθήστε μας να επιλύσουμε τυχόν προβλήματα και να βελτιώσουμε τα προϊόντα και τις υπηρεσίες της Microsoft. Στείλτε στη Microsoft διαγνωστικά δεδομένα (καθώς και δεδομένα σχετικά με τη χρήση προγραμμάτων περιήγησης, εφαρμογών και δυνατοτήτων και δεδομένα γραφής και πληκτρολόγησης).

Στο βασικό επίπεδο, θα στέλνετε στη Microsoft λιγότερα δεδομένα για την επιδιόρθωση των σφαλμάτων που παρουσιάζονται.

ΕνεργόΑνενεργό
Εξατομικευμένες εμπειρίες με τη βοήθεια διαγνωστικών δεδομένων

Λάβετε πιο σχετικές συμβουλές και προτάσεις για την προσαρμογή των προϊόντων και των υπηρεσιών της Microsoft στις ανάγκες σας. Για να γίνει αυτό, επιτρέψτε στη Microsoft να χρησιμοποιεί τα διαγνωστικά δεδομένα σας.

Οι συμβουλές που θα λαμβάνετε θα είναι πιο γενικές και οι προτάσεις ενδέχεται να μην είναι ιδιαίτερα εξατομικευμένες.

ΕνεργόΑνενεργό
Σχετικές διαφημίσεις

Επιτρέψτε επιτρέπεται στις εφαρμογές να χρησιμοποιούν το αναγνωριστικό διαφήμισης για την παροχή διαφημίσεων που παρουσιάζουν μεγαλύτερο ενδιαφέρον για εσάς με βάση τη χρήση εφαρμογών.

Το πλήθος των διαφημίσεων που θα βλέπετε δεν θα αλλάξει, αλλά μπορεί να μην είναι ιδιαίτερα εξατομικευμένες.

Επιλέξτε Μάθετε περισσότερα για πληροφορίες σχετικά με τις παραπάνω ρυθμίσεις, τον τρόπο λειτουργίας του Windows Defender SmartScreen και τις σχετικές μεταφορές και χρήσεις δεδομένων.​

Ευχαριστούμε! Αποθηκεύσαμε τις ρυθμίσεις προστασίας προσωπικών δεδομένων σας για την επόμενη ενημέρωση

Η ενημέρωση δεν θα ξεκινήσει όταν επιλέξετε το OK. Θα εκτελεστούν πρώτα ορισμένες εργασίες στο παρασκήνιο και έπειτα θα σας ειδοποιήσουμε πριν από την έναρξη της ενημέρωσης. Αν θα θέλατε να τεθεί σε αναμονή, επιλέξτε Όχι τώρα και θα σας το υπενθυμίσουμε ξανά αργότερα.
